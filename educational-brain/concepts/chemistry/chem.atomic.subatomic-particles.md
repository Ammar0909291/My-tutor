## Learning Objective

Students can state the charge, relative mass, and location of proton, neutron, and electron; define atomic number (Z) and mass number (A); write isotope notation; distinguish isotopes, isobars, and isotones by the quantity held constant; and calculate the number of each subatomic particle from isotope notation.

## Core Understanding

Every atom of an element is defined by its **atomic number (Z)** — the number of protons in the nucleus. This is the invariant identity of the element: change the proton count and you have a different element. The **mass number (A)** = protons + neutrons. The neutron count = A − Z.

**Isotopes** are atoms of the same element (same Z) with different neutron numbers (different A). They are chemically almost identical (same electron count → same chemical behaviour) but physically different (different mass → different densities, nuclear stability).

**Isobars** share the same mass number (A) but different atomic numbers (Z) — different elements.

**Isotones** share the same neutron number (A − Z) but different Z — different elements with different mass numbers.

The electron count in a neutral atom equals Z. In an ion, the electron count deviates from Z by the ion's charge (positive ion = lost electrons; negative ion = gained electrons).

## Mental Models

**The three-property address**: Every atomic species has a complete address: Z (proton count = element identity), N (neutron count = isotope identity), charge (electron count = ion identity). Specify all three and the species is fully identified.

**Nuclear vs. outer electron distinction**: The nucleus (protons + neutrons) is the heavy, dense, positively charged core — chemically inert in normal reactions. The electrons orbit in the vast surrounding space and govern all chemical behaviour.

## Why Students Fail

1. **Confusing mass number with atomic mass**: Mass number (A) is always a whole number (count of nucleons); atomic mass shown on the periodic table is a decimal (weighted average of isotope masses). Students use them interchangeably.
2. **Neutron count errors**: Students set N = A instead of N = A − Z, forgetting to subtract protons.
3. **Isotopes vs. different elements**: Students think isotopes are different elements because their masses differ; they don't connect same-Z = same-element firmly enough.
4. **Ion electron counting**: Students forget that a positive ion has fewer electrons than protons, and a negative ion has more.

## Misconceptions

**MC-1 — Mass number = atomic mass** (Type 4, notation-induced)
- *Diagnostic probe*: "Chlorine's atomic mass is 35.45. Is there a chlorine isotope with mass number 35.45?"
- *Characteristic phrase*: "The atomic mass is 35.45, so the mass number is 35.45."
- *Repair*: Mass number counts whole protons and neutrons (always an integer). Atomic mass is a weighted average across the abundances of all naturally occurring isotopes. ³⁵Cl (mass number 35) and ³⁷Cl (mass number 37) together, weighted 75.77% and 24.23%, give 35.45 as the average.

**MC-2 — Isotopes are different elements** (Type 3, language contamination)
- *Diagnostic probe*: "¹²C and ¹⁴C — are these the same element? How do you know? What is the same? What is different?"
- *Characteristic phrase*: "C-12 and C-14 are different elements because their masses are different."
- *Repair*: The element identity is Z only. Both have Z = 6 (six protons) → both are carbon. The neutron difference (6 vs. 8) affects mass and nuclear stability but not chemical identity.

**MC-3 — Ion electron count equals proton count** (Type 5, instruction-induced)
- *Diagnostic probe*: "A sodium ion Na⁺ has how many electrons? A chloride ion Cl⁻ has how many electrons?"
- *Characteristic phrase*: "Na⁺ has 11 electrons because sodium has 11 protons."
- *Repair*: Na⁺ lost one electron → 11 − 1 = 10 electrons. Cl⁻ gained one electron → 17 + 1 = 18 electrons. The charge tells you the electron adjustment.

## Analogies

**Isotopes as different editions of the same book**: The content (chemical behaviour → electron arrangement) is the same; the printing (mass, page weight → neutron count) differs.

## Demonstrations

**Demonstration 1 — Isotope table exercise**
- Present a table of chlorine isotopes (³⁵Cl: 75.77%; ³⁷Cl: 24.23%; ³⁶Cl: trace). Students calculate Z, N, and electron count for each. Then they compute the weighted-average atomic mass: (35 × 0.7577) + (37 × 0.2423) = 35.48 g/mol (≈ the tabulated 35.45, with rounding). This cements the distinction between mass number and atomic mass.

## Discovery Questions

1. "If you add a neutron to a ¹H atom, what do you get? Add another. What are the names of these species?" (Targets: deuterium ²H, tritium ³H — isotopes of hydrogen, the most famous example.)
2. "²⁴Mg and ²⁴Na — are these isotopes, isobars, or isotones? What does each classification tell you?" (Targets: same A (24) but different Z → isobars.)
3. "Why do chemists care that ¹⁴C undergoes radioactive decay but ¹²C does not, even though both are carbon?" (Targets: different neutron-to-proton ratios → nuclear stability; ¹⁴C dating application.)

## Teaching Sequence

1. Review from chem.atomic.atomic-theory: nuclear model, three-particle picture.
2. Present the properties table: charge, mass, location for proton, neutron, electron. Students copy and verify against Rutherford's model.
3. Define Z and A. Work examples: given an isotope symbol, read Z and A, then compute N, proton count, neutron count, electron count (for neutral atom and for given ion).
4. Define isotopes, isobars, isotones. Give three pairs and ask students to classify each.
5. Run Demonstration 1 (chlorine isotope weighted average). This simultaneously reinforces isotope structure and distinguishes mass number from atomic mass.
6. Ion electron counting: practice with 6 ions (3 cations, 3 anions).

## Tutor Actions

- If student uses decimal atomic mass as the mass number → MC-1 repair: the weighted average exercise.
- If student says isotopes are different elements → MC-2 repair: same-Z = same element, always.
- If student miscomputes ion electron count → MC-3 repair: charge = electron adjustment; write out the subtraction/addition explicitly.
- Advance when student can, given any isotope notation, correctly state Z, A, N, proton count, neutron count, and electron count (neutral and ionic).

## Voice Teaching Notes

"Z is the element's identity card number" — repeat this phrase each time atomic number is introduced. It links Z to the permanent identity of the element more concretely than "atomic number."

The proton/neutron/electron properties table should be drawn from memory by the student at the start of the next session (retrieval practice) before any new material.

## Assessment Signals

**Mastery gate**:
1. Given the notation ⁵⁶Fe²⁺, student correctly states: Z = 26, A = 56, N = 30, protons = 26, neutrons = 30, electrons = 24.
2. Student correctly classifies three pairs of species as isotopes, isobars, or isotones.
3. Student explains why ¹²C and ¹⁴C are the same element but have different masses.

**FRAGILE signal**: student reads Z and A correctly but computes N = A (not A − Z).

**MISCONCEIVING signal**: student says decimal atomic mass is the mass number — requires the weighted-average exercise immediately.

## Tutor Recovery Strategy

If stuck:
1. Return to the three-part address: Z, N, charge. Ask the student to read each from the isotope symbol step by step, writing each value separately before any calculation.
2. For isotope/element confusion: emphasise Z as the sole determinant of element identity by going through the periodic table — every box's number is Z, not A.
3. For ion electrons: write out "protons − charge = electrons" as a formula. For cations: subtract charge. For anions: add |charge| (because the charge is negative).

## Memory Hooks

- **Particle properties**: "Proton: +1, 1u, nucleus. Neutron: 0, 1u, nucleus. Electron: −1, ~0u, orbital."
- **N = A − Z**: "Neutrons = mass minus atomic." One sentence.
- **Isotopes same Z; isobars same A; isotones same N.** Three pairs, one distinctive variable each.
