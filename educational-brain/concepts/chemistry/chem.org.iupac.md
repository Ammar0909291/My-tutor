## Learning Objective
Apply IUPAC systematic nomenclature rules to name and draw structures for alkanes, alkenes, alkynes, and their common functional group derivatives (halides, alcohols, aldehydes, ketones, carboxylic acids, esters, amines, amides) up to 10 carbon chain length, including branched and cyclic structures.

## Core Understanding
IUPAC nomenclature is a systematic rule set for giving an unambiguous name to every organic compound. The name encodes the structure; the structure can be exactly reconstructed from the name. A systematic IUPAC name consists of up to four parts: **substituent prefixes + parent chain root + principal characteristic group suffix + multiplying infix (en, yn for unsaturation)**.

**Parent chain selection rules** (in order of priority):
1. Longest carbon chain that includes the principal characteristic group (highest-priority functional group).
2. Maximum number of multiple bonds.
3. Lowest set of locants for substituents (when summed).
4. Alphabetical order for identical-position substituents.

**Chain length prefixes (root names)**:
meth- (1), eth- (2), prop- (3), but- (4), pent- (5), hex- (6), hept- (7), oct- (8), non- (9), dec- (10).

**Functional group suffix priority (highest first — only one suffix per name)**:
-oic acid (carboxylic acid, RCOOH) > -al (aldehyde, RCHO) > -one (ketone, RCOR') > -ol (alcohol, ROH) > -amine (amine, RNH₂)

When a higher-priority group is present, lower-priority groups appear as prefixes: oxo- (for =O on a chain, when there is a higher-priority suffix), hydroxy-, amino-.

**Naming procedure**:
1. Identify the principal characteristic group (highest in priority order); this sets the suffix.
2. Find the longest chain including the principal group.
3. Number from the end closest to the principal group (giving it the lowest locant).
4. If numbering is a tie, give the lowest locant set to substituents.
5. Name substituents as prefixes in alphabetical order (ignoring multiplying prefixes di-, tri-).
6. Indicate unsaturation with -en- (double bond) or -yn- (triple bond) plus locant.
7. Assemble: (locant-substituent prefix) + parent root + (unsaturation infix) + suffix with locant.

**Cyclic compounds**: prefix cyclo- before the parent root; numbering follows the same lowest-locant rule for the ring.

**Branched substituent naming**: alkyl groups as substituents are named as prefix; methyl-, ethyl-, propyl-, isopropyl- (= 1-methylethyl-), butyl-, sec-butyl- (= 1-methylpropyl-), tert-butyl- (= 1,1-dimethylethyl-), isobutyl- (= 2-methylpropyl-).

**Worked example**: CH₃–CH(Cl)–CH₂–CH₂–COOH
- Principal group: –COOH → suffix -oic acid
- Longest chain including –COOH: 5 C → pentanoic acid
- Cl substituent at C4 (numbering from –COOH end gives Cl at C4; from other end gives C2); lowest locant → number from –COOH: Cl is at C4
- Name: 4-chloropentanoic acid

**Common name cross-references** (expected to recognise at minimum):
- Acetic acid = ethanoic acid; formic acid = methanoic acid; acetone = propanone; ethylene = ethene; acetylene = ethyne; aniline = aminobenzene (benzenamine by IUPAC).

## Mental Models
**The address model**: an IUPAC name is like a postal address — country (suffix = functional class), city (parent root = chain length), street number (locants). A complete, unambiguous address has all parts; a vague name is like a city name without a street number.

**The parent chain skeleton**: visualise the longest chain as a backbone; every other carbon or group hangs off as a branch; the name first names the backbone, then the branches in alphabetical order, then gives every branch a street address (locant).

## Why Students Fail
Students select the longest carbon chain regardless of whether it includes the principal functional group — a chain that includes the –COOH but is shorter must be chosen over a longer chain that does not.

Students number from the wrong end — always number from the end that gives the principal characteristic group the LOWEST locant first, then minimise substituent locants.

Students confuse suffix and prefix for the same functional group — alcohols are -ol as a suffix when they are the highest-priority group, but hydroxy- as a prefix when a higher-priority group is also present.

## Misconceptions
- **MC-1 (Type 5 — instruction-induced)**: "Always choose the longest carbon chain, regardless of functional groups." Probe: name the compound CH₃–CH₂–COOH with a –CH₂–CH₂–CH₃ branch on the C2 carbon. Characteristic phrase: "I chose the 6-carbon chain." Intervention: the parent chain must INCLUDE the –COOH group; the chain containing the principal characteristic group takes priority over chain length.
- **MC-2 (Type 4 — notation-induced)**: "Locants are written after the substituent name (e.g., methyl-2 instead of 2-methyl)." Probe: "Correct the name 'pentanone-3'." Characteristic phrase: "I put the number after the word." Intervention: IUPAC convention places the locant immediately before the part of the name it modifies (2-methylpentane; pentan-3-one or 3-pentanone in older nomenclature; 2-methyl, not methyl-2); current IUPAC recommends locant immediately before the modified element.
- **MC-3 (Type 1 — overgeneralization)**: "The suffix -one always means ketone." Probe: "What is cyclohexanone's principal group?" Characteristic phrase: "any -one must be a ketone." Intervention: -one is the ketone suffix only when the C=O is NOT at the chain terminus; in a ring it may still be a ketone (cyclohexanone is correct); but confusing -one with -al (aldehyde) for terminal C=O is the common error.
- **MC-4 (Type 5 — instruction-induced)**: "Alphabetical ordering of substituents includes di-, tri- multiplying prefixes." Probe: "In 3-ethyl-2,4-dimethylhexane, does 'di' count when alphabetising?" Intervention: multiplying prefixes (di-, tri-, tetra-) are NOT included in alphabetising; ethyl comes before methyl (e < m), regardless of the 'di' prefix on methyl.

## Analogies
**Valid**: IUPAC nomenclature is like a recipe card index system — there is exactly one correct card for every compound, and the name tells you which drawer (functional class), which section (chain length), and which card number (locants and substituents) to pull out.

**Anti-analogy**: Do NOT treat common names (acetone, acetic acid) as equivalent alternatives in systematic work — they are shortcuts for specific well-known compounds, not a parallel system; they cannot be extended to unfamiliar compounds.

## Demonstrations
**Name-to-structure drill**: give students five IUPAC names and ask them to draw the structural formula. Then give five structural formulas and ask for the IUPAC name. Swap papers and check in pairs. Immediate bi-directional practice embeds both skills.

**Systematic vs. common name recognition**: give the five most common trivial names (acetone, acetic acid, formaldehyde, ethylene glycol, formic acid) and systematically derive the IUPAC equivalent from the structure. Emphasises that common names can only be RECOGNISED, not generated systematically.

## Discovery Questions
1. "A compound has the structure: CH₃–C(=O)–CH(CH₃)–CH₂–CH₂–COOH. Work through the four-step naming procedure to give the full IUPAC name."
2. "Draw the structural formula for (E)-3-chlorobut-2-enoic acid." (Requires applying unsaturation locant + stereodescriptor + halide prefix + acid suffix simultaneously.)
3. "Why does IUPAC discourage using 'acetyl' as a prefix for CH₃CO– in systematic names?"

## Teaching Sequence
1. **Establish the four-part name structure**: suffix, root, infix, prefix — show an example name assembled from its parts.
2. **Parent chain selection**: practice with unbranched alkanes first; confirm longest-chain rule; then introduce the functional-group inclusion priority.
3. **Numbering rules**: lowest locant for principal group; then lowest locant set for substituents; practice with three exercises.
4. **Functional group suffixes**: work through the priority list; practise converting between structure and name for one functional class at a time (acid → aldehyde → ketone → alcohol → amine).
5. **Substituent prefixes**: common alkyl groups; alphabetical ordering (excluding di-/tri-); use multiplying prefixes for identical substituents.
6. **Unsaturation**: locate double and triple bonds; use -en- and -yn- infixes; locant rules.
7. **Cyclic compounds**: prefix cyclo-; numbering.
8. **Integration exercises**: 5 complex structures named; 5 complex names drawn.

## Tutor Actions
- **If student consistently names wrong chain length**: ask "does the chain you chose include the –COOH/–OH/principal group?" — if not, reject the chain and find the longest one that does.
- **If student places locant at the wrong end**: ask "what is the lowest number the principal group can have?" — numbering must minimise the principal group's locant first; check both directions of numbering explicitly.
- **If student alphabetises with 'di-' included**: demonstrate with a specific example — "3-ethyl-2,4-dimethylhexane: which comes first alphabetically, ethyl or methyl?" (ethyl, ignoring 'di'); the multiplying prefix is transparent for alphabetising.
- **FRAGILE sign**: names simple unbranched compounds correctly but makes chain-selection or locant errors in branched multi-functional compounds.

## Voice Teaching Notes
Naming is primarily a procedural skill — the tutor should narrate the steps aloud and ask the student to narrate back for the next example. "What is the principal group? What is the longest chain? Which end do I number from?" as a spoken checklist enforces the procedure.

Students who cannot reconstruct the procedure from memory after correct answers should be asked to do it on a new example without prompting — transfer to novel structures is the mastery test, not recognition.

For adult learners returning to chemistry: emphasise that IUPAC names were not used historically (all older literature uses trivial names); knowing both is important for reading both modern and classic texts.

## Assessment Signals
- **Green**: correctly names and draws 5 multi-functional, branched structures; applies the locant and alphabetical ordering rules correctly on novel examples.
- **Amber**: correct procedure for simple chains but makes errors in multi-functional or branched cases; or names in one direction (name from structure) but cannot reverse (structure from name).
- **Red**: always picks the longest chain regardless of functional group; numbers from the wrong end; cannot recall chain-length root words above 4 carbons.
- **Probe**: "Give the IUPAC name for: CH₃–C(CH₃)(OH)–CH₂–CH=CH₂." (Expected: 2-methylpent-4-en-2-ol — longest chain including –OH and double bond = 5 carbons; number from –OH end; –OH at C2, double bond at C4.)

## Tutor Recovery Strategy
If student cannot apply the four-step procedure reliably: strip the problem entirely. Start with unbranched alkanes only (no functional groups, no branches); confirm chain length root name → apply one rule at a time, adding branches in step 2, functional groups in step 3, then unsaturation in step 4. Do NOT combine multiple rules in a single example until each rule is independently reliable.

## Memory Hooks
- **Root names**: "Methy, Eth, Prop, But, Pent, Hex, Hept, Oct, Non, Dec" — memorised as a count (1 through 10).
- **Priority order for suffix**: "CAKE — Carboxylic acid, Aldehyde, Ketone, Ester (then alcohol, amine lower)" — highest priority gets the suffix.
- **Alphabetical ordering**: "ignore the multiplying prefix" — eth comes before meth before prop; di-/tri-/tetra- are transparent.
- **Number from the principal group's end first**: "Give the functional group the lowest address number."

## Cross-Subject Connections
- **Pharmacy (drug naming)**: IUPAC names are used in drug monographs and pharmacopeias alongside trade names; understanding systematic names enables accurate identification of drug structures.
- **Materials science (polymer naming)**: IUPAC polymer nomenclature extends the same chain-naming principles to repeating unit names (poly(ethylene), poly(propylene)) — the chain root names transfer directly.
