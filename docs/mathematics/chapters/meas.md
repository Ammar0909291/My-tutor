## Sigma-Algebras

**Concept ID:** `math.meas.sigma-algebra`  
**Prerequisites:** `math.found.set-theory`  
**Difficulty:** 3

A **sigma-algebra** (σ-algebra) on a set X is a collection Σ of subsets of X satisfying three axioms:
1. ∅ ∈ Σ
2. A ∈ Σ ⟹ Aᶜ ∈ Σ (closed under complementation)
3. A₁, A₂, ... ∈ Σ ⟹ ⋃ₙAₙ ∈ Σ (closed under countable unions)

The pair (X, Σ) is called a **measurable space**. Sigma-algebras define which subsets of X are "measurable" — the admissible domain for a measure.

### Key Examples

- **Trivial σ-algebra:** {∅, X}
- **Power set:** 2^X (every subset is measurable)
- **Borel σ-algebra B(ℝ):** generated by all open intervals; the canonical choice for analysis on ℝ

### Generated Sigma-Algebras

The σ-algebra generated by a collection 𝒞, denoted σ(𝒞), is the smallest σ-algebra containing 𝒞. It is always well-defined as the intersection of all σ-algebras containing 𝒞.

### Why Countable, Not Uncountable?

The restriction to countable unions is essential: it prevents measure-theoretic paradoxes and ensures that measure can be consistently defined. Uncountable unions of measurable sets need not be measurable.

## Measures

**Concept ID:** `math.meas.measure`  
**Prerequisites:** `math.meas.sigma-algebra`  
**Difficulty:** 3

A **measure** μ on a measurable space (X, Σ) is a function μ: Σ → [0, ∞] satisfying:
1. μ(∅) = 0
2. **Countable additivity:** if {Aₙ} ⊆ Σ are pairwise disjoint, then μ(⋃ₙAₙ) = Σₙμ(Aₙ)

The triple (X, Σ, μ) is called a **measure space**.

### Fundamental Properties

Derived from the axioms:

- **Monotonicity:** A ⊆ B ⟹ μ(A) ≤ μ(B)
- **Subadditivity:** μ(⋃ₙAₙ) ≤ Σₙμ(Aₙ)
- **Continuity from below:** Aₙ ↑ A ⟹ μ(Aₙ) ↑ μ(A)
- **Continuity from above:** Aₙ ↓ A and μ(A₁) < ∞ ⟹ μ(Aₙ) ↓ μ(A)
- **Inclusion-exclusion:** μ(A ∪ B) = μ(A) + μ(B) − μ(A ∩ B)

### Key Examples

| Measure | Space | Definition |
|---------|-------|------------|
| Lebesgue measure λ | (ℝ, B(ℝ)) | λ([a,b]) = b − a |
| Counting measure | (ℕ, 2^ℕ) | μ(S) = \|S\| |
| Dirac measure δₓ | (X, Σ) | δₓ(A) = 1 if x ∈ A, else 0 |
| Probability measure P | (Ω, Σ) | P(Ω) = 1 |

### Classification

- **Finite:** μ(X) < ∞
- **σ-finite:** X = ⋃ₙXₙ with μ(Xₙ) < ∞ (most theorems require this)
- **Probability:** μ(X) = 1

## Lebesgue Measure

**Concept ID:** `math.meas.lebesgue-measure`  
**Prerequisites:** `math.meas.measure`  
**Difficulty:** 4

The **Lebesgue measure** λ is the unique translation-invariant Borel measure on ℝ with λ([a,b]) = b − a.

### Construction via Outer Measure

**Step 1 — Lebesgue outer measure:**
$$\lambda^*(E) = \inf\left\{ \sum_n |I_n| : E \subseteq \bigcup_n I_n,\ I_n \text{ open intervals} \right\}$$

**Step 2 — Carathéodory measurability:** E is Lebesgue measurable if for all A ⊆ ℝ:
$$\lambda^*(A) = \lambda^*(A \cap E) + \lambda^*(A \cap E^c)$$

**Step 3:** Restrict λ* to the class of measurable sets — this gives the Lebesgue measure.

### Key Properties

- **Translation invariance:** λ(E + t) = λ(E) for all measurable E, t ∈ ℝ
- **Scaling:** λ(cE) = |c|·λ(E)
- **Countable sets have measure zero:** E countable ⟹ λ(E) = 0
- The Cantor set is uncountable with λ(Cantor) = 0
- Non-measurable sets (Vitali sets) exist — not all subsets of ℝ are Lebesgue measurable

### Relationship to Borel σ-algebra

Every Borel set is Lebesgue measurable, but the Lebesgue σ-algebra is strictly larger (it is the completion of the Borel σ-algebra).

## Sets of Measure Zero

**Concept ID:** `math.meas.measure-zero`  
**Prerequisites:** `math.meas.lebesgue-measure`  
**Difficulty:** 2

A measurable set N has **measure zero** (is a **null set**) if μ(N) = 0.

### Almost Everywhere

A property P holds **almost everywhere** (a.e.) if the set where P fails is a null set:
$$\{x : P(x) \text{ fails}\}\ \text{has measure zero}$$

### Properties of Null Sets

- Every countable set in ℝ has Lebesgue measure zero
- The Cantor set: uncountable, yet λ(Cantor) = 0 — measure zero ≠ "small" topologically
- **Countable union:** if μ(Nₙ) = 0 for all n, then μ(⋃ₙNₙ) = 0 (by countable subadditivity)
- **Uncountable union:** no longer guaranteed null — ⋃_{x∈ℝ}{x} = ℝ has measure ∞

### Why Almost Everywhere Matters

The Lebesgue integral is blind to null sets: ∫f dμ = ∫g dμ whenever f = g a.e. This makes L^p spaces well-defined (as equivalence classes under a.e. equality) and enables powerful convergence theorems.

### The Dirichlet Function

The function f(x) = 1_{ℚ}(x) (1 on rationals, 0 on irrationals) satisfies:
- Not Riemann integrable
- f = 0 a.e. (since λ(ℚ) = 0)  
- Lebesgue integral: ∫₀¹ f dλ = 0

## Measurable Functions

**Concept ID:** `math.meas.measurable-function`  
**Prerequisites:** `math.meas.sigma-algebra`  
**Difficulty:** 3

A function f: (X, Σ) → (Y, Γ) is **measurable** if pre-images of measurable sets are measurable:
$$\forall B \in \Gamma:\ f^{-1}(B) \in \Sigma$$

### Equivalent Characterisations (for f: X → ℝ)

The following are equivalent for measurability with respect to B(ℝ):
- {x: f(x) > a} ∈ Σ for all a ∈ ℝ
- {x: f(x) ≥ a} ∈ Σ for all a ∈ ℝ
- {x: f(x) < a} ∈ Σ for all a ∈ ℝ
- {x: f(x) ≤ a} ∈ Σ for all a ∈ ℝ

### Stability Properties

If f, g are measurable, so are:
- f + g, f − g, fg, f/g (where g ≠ 0)
- |f|, f⁺ = max(f, 0), f⁻ = max(−f, 0)
- max(f, g), min(f, g)
- lim supₙ fₙ and lim infₙ fₙ (for sequences)
- The pointwise limit f = lim fₙ (if it exists)

### Key Examples

- Every continuous function ℝ → ℝ is Borel measurable
- Every monotone function ℝ → ℝ is Borel measurable
- The Dirichlet function 1_{ℚ} is Borel measurable (ℚ is a countable Borel set)

## Simple Functions

**Concept ID:** `math.meas.simple-function`  
**Prerequisites:** `math.meas.measurable-function`  
**Difficulty:** 3

A **simple function** is a measurable function taking finitely many values. In **canonical form**:
$$s = \sum_{i=1}^n c_i \cdot \mathbf{1}_{A_i}$$
where c₁,...,cₙ are distinct non-negative reals and {A₁,...,Aₙ} is a measurable partition of X.

### Integral of a Simple Function

$$\int s \, d\mu = \sum_{i=1}^n c_i \cdot \mu(A_i) \quad (\text{convention: } 0 \cdot \infty = 0)$$

### The Approximation Theorem

For every measurable f ≥ 0, there exists an increasing sequence sₙ ↑ f pointwise, where each sₙ is a simple function. One standard construction:

$$s_n(x) = \frac{k}{2^n} \text{ if } \frac{k}{2^n} \le f(x) < \frac{k+1}{2^n},\ k = 0, \ldots, n \cdot 2^n - 1$$
$$s_n(x) = n \text{ if } f(x) \ge n$$

This approximation is the cornerstone of the Lebesgue integral definition.

## The Lebesgue Integral

**Concept ID:** `math.meas.lebesgue-integral`  
**Prerequisites:** `math.meas.simple-function`  
**Difficulty:** 4

### Three-Stage Definition

**Stage 1 — Simple functions:** ∫s dμ = Σᵢ cᵢ μ(Aᵢ)

**Stage 2 — Non-negative measurable functions:**
$$\int f \, d\mu = \sup\left\{ \int s \, d\mu : 0 \le s \le f,\ s \text{ simple} \right\}$$

**Stage 3 — General integrable functions:** Write f = f⁺ − f⁻ and set
$$\int f \, d\mu = \int f^+ \, d\mu - \int f^- \, d\mu$$
provided at least one of the integrals is finite. f is **integrable** if ∫|f| dμ < ∞.

### Key Properties

| Property | Statement |
|----------|-----------|
| Linearity | ∫(αf + βg) dμ = α∫f dμ + β∫g dμ |
| Monotonicity | f ≤ g a.e. ⟹ ∫f dμ ≤ ∫g dμ |
| Null-set insensitivity | f = g a.e. ⟹ ∫f dμ = ∫g dμ |
| Triangle inequality | \|∫f dμ\| ≤ ∫\|f\| dμ |

### Comparison with Riemann Integral

- **Agreement:** for Riemann-integrable f, both integrals agree
- **Extension:** the Lebesgue integral applies to a strictly larger class (e.g., the Dirichlet function: ∫₀¹ 1_{ℚ} dλ = 0)
- **Conceptual difference:** Riemann partitions the domain; Lebesgue partitions the range

## Convergence Theorems

**Concept ID:** `math.meas.convergence-theorems`  
**Prerequisites:** `math.meas.lebesgue-integral`  
**Difficulty:** 4

These three theorems characterise when lim ∫fₙ dμ = ∫lim fₙ dμ.

### Monotone Convergence Theorem (MCT)

**Hypothesis:** 0 ≤ f₁ ≤ f₂ ≤ ... measurable, fₙ → f a.e.  
**Conclusion:** ∫fₙ dμ → ∫f dμ  
**Note:** Allows ∞; no domination needed.

### Fatou's Lemma

**Hypothesis:** fₙ ≥ 0 measurable  
**Conclusion:** ∫(lim inf fₙ) dμ ≤ lim inf ∫fₙ dμ  
**Note:** Inequality can be strict.

### Dominated Convergence Theorem (DCT)

**Hypothesis:** fₙ → f a.e., |fₙ| ≤ g a.e. with ∫g dμ < ∞  
**Conclusion:** fₙ, f ∈ L¹ and ∫fₙ dμ → ∫f dμ (also ∫|fₙ − f| dμ → 0)

### The Essential Counterexample (Why Domination Matters)

Let fₙ(x) = n · 1_{[0,1/n]}(x). Then:
- fₙ → 0 pointwise a.e.
- ∫fₙ dλ = 1 for all n

So ∫fₙ → 1 ≠ 0 = ∫lim fₙ. The spike functions have no integrable dominator.

## L^p Spaces

**Concept ID:** `math.meas.lp-space`  
**Prerequisites:** `math.meas.lebesgue-integral`  
**Difficulty:** 4

For a measure space (X, Σ, μ) and 1 ≤ p < ∞:
$$L^p(X, \mu) = \left\{ f \text{ measurable} : \int |f|^p \, d\mu < \infty \right\} / \sim$$
where f ~ g iff f = g a.e., with norm ‖f‖_p = (∫|f|^p dμ)^{1/p}.

For p = ∞: ‖f‖_∞ = ess sup|f| = inf{M : |f| ≤ M a.e.}.

### Fundamental Inequalities

**Hölder's inequality** (conjugate exponents: 1/p + 1/p' = 1):
$$\|fg\|_1 \le \|f\|_p \cdot \|g\|_{p'}$$

**Minkowski's inequality** (triangle inequality for L^p):
$$\|f + g\|_p \le \|f\|_p + \|g\|_p$$

### Banach Space Structure

The **Riesz-Fischer theorem** states L^p is complete — every Cauchy sequence in L^p converges to an element of L^p. Hence L^p is a **Banach space**.

### L^p Inclusions

On **finite measure spaces** (e.g., [0,1]): p < q ⟹ L^q ⊆ L^p  
On **sequence spaces** (counting measure): ℓ^p ⊆ ℓ^q for p ≤ q

## L^2 Space and Hilbert Space Structure

**Concept ID:** `math.meas.l2-space`  
**Prerequisites:** `math.meas.lp-space`  
**Difficulty:** 4

L^2(X, μ) is a **Hilbert space** with inner product:
$$\langle f, g \rangle = \int f \cdot \bar{g} \, d\mu$$

The induced norm is ‖f‖₂ = ⟨f,f⟩^{1/2}, consistent with the L^p norm for p = 2.

### Hilbert Space Geometry

- **Cauchy-Schwarz:** |⟨f,g⟩| ≤ ‖f‖₂ · ‖g‖₂
- **Orthogonality:** f ⊥ g iff ⟨f,g⟩ = 0
- **Orthogonal projection:** every closed subspace V has a unique projection P_V with ‖f − P_V f‖ = inf_{v∈V}‖f − v‖

### Orthonormal Bases and Parseval's Identity

For any orthonormal basis (ONB) {eₙ}:
$$f = \sum_n \langle f, e_n \rangle e_n \quad (\text{convergence in } L^2)$$
$$\|f\|_2^2 = \sum_n |\langle f, e_n \rangle|^2 \quad (\text{Parseval's identity})$$

The trigonometric system {e^{inx}/√(2π) : n ∈ ℤ} is an ONB for L^2([−π,π]) — Fourier series.

### Riesz Representation Theorem

Every bounded linear functional Λ on L^2 has a unique g ∈ L^2 with Λ(f) = ⟨f,g⟩ and ‖Λ‖ = ‖g‖₂. This identifies (L^2)* with L^2 itself — L^2 is self-dual.

## Product Measures and Fubini's Theorem

**Concept ID:** `math.meas.product-measure`  
**Prerequisites:** `math.meas.lebesgue-integral`  
**Difficulty:** 4

Given measure spaces (X, Σ, μ) and (Y, Γ, ν), the **product measure** μ ⊗ ν on the **product σ-algebra** Σ ⊗ Γ = σ({A × B : A ∈ Σ, B ∈ Γ}) is the unique measure satisfying:
$$(μ \otimes ν)(A \times B) = μ(A) \cdot ν(B)$$

### Tonelli's Theorem (non-negative functions)

If f: X × Y → [0,∞] is measurable, then:
$$\int_{X \times Y} f \, d(\mu \otimes \nu) = \int_X \left( \int_Y f(x,y) \, d\nu(y) \right) d\mu(x) = \int_Y \left( \int_X f(x,y) \, d\mu(x) \right) d\nu(y)$$

All three expressions are equal (possibly +∞). No integrability required.

### Fubini's Theorem (integrable functions)

If ∫|f| d(μ ⊗ ν) < ∞, then:
- For μ-a.e. x, the section y ↦ f(x,y) is ν-integrable
- For ν-a.e. y, the section x ↦ f(x,y) is μ-integrable
- The iterated integrals equal the double integral (same conclusion as Tonelli)

### Counterexample Without Integrability

f(x,y) = (x² − y²)/(x² + y²)² on [0,1] × [0,1]:
- ∫₀¹(∫₀¹ f dx) dy = π/4 ≠ −π/4 = ∫₀¹(∫₀¹ f dy) dx

## The Radon-Nikodym Theorem

**Concept ID:** `math.meas.radon-nikodym`  
**Prerequisites:** `math.meas.lebesgue-integral`  
**Difficulty:** 5

### Absolute Continuity

Measure ν is **absolutely continuous** with respect to μ (written ν ≪ μ) if:
$$\mu(E) = 0 \implies \nu(E) = 0 \quad \text{for all } E \in \Sigma$$

### The Radon-Nikodym Theorem

If μ and ν are σ-finite and ν ≪ μ, then there exists a unique (μ-a.e.) non-negative measurable function f = dν/dμ such that:
$$\nu(E) = \int_E \frac{d\nu}{d\mu} \, d\mu \quad \text{for all } E \in \Sigma$$

The function dν/dμ is the **Radon-Nikodym derivative** (or density) of ν with respect to μ.

### Change of Variables Formula

$$\int g \, d\nu = \int g \cdot \frac{d\nu}{d\mu} \, d\mu$$

### Chain Rule

If ρ ≪ ν ≪ μ, then dρ/dμ = (dρ/dν) · (dν/dμ) a.e.

### Lebesgue Decomposition

Any σ-finite ν decomposes uniquely as ν = νₐ + νₛ where νₐ ≪ μ (absolutely continuous part) and νₛ ⊥ μ (singular part: supported on a μ-null set).

### Probabilistic Interpretation

If P ≪ λ (Lebesgue), the Radon-Nikodym derivative f = dP/dλ is the **probability density function**. Conditional expectation E[X | 𝒢] is the Radon-Nikodym derivative of E ↦ E[X · 1_E] with respect to P|_𝒢.

## Abstract Measure Spaces

**Concept ID:** `math.meas.abstract-measure-spaces`  
**Prerequisites:** `math.meas.measure`  
**Difficulty:** 4

### Classification of Measure Spaces

| Property | Definition |
|----------|------------|
| Finite | μ(X) < ∞ |
| σ-finite | X = ⋃ₙXₙ with μ(Xₙ) < ∞ |
| Complete | Every subset of a null set is measurable |
| Probability | μ(X) = 1 |
| Regular | Inner and outer regular (for topological spaces) |

### The Carathéodory Extension Theorem

Starting from a **premeasure** μ₀ on a **ring** ℛ (closed under finite set operations), the Carathéodory theorem extends μ₀ uniquely to a measure on σ(ℛ), provided μ₀ is σ-finite.

**Construction path for Lebesgue measure:**  
Length function on intervals → premeasure on ring of finite unions of half-open intervals → Carathéodory extension → Lebesgue measure on B(ℝ)

### Completion of a Measure Space

The **completion** (X, Σ̄, μ̄) adds all subsets of null sets:
$$\bar{\Sigma} = \{A \cup N : A \in \Sigma,\ N \subseteq M \in \Sigma,\ \mu(M) = 0\}$$

The Lebesgue σ-algebra is the completion of the Borel σ-algebra under the Lebesgue measure.

### Regular Measures (on locally compact Hausdorff spaces)

- **Inner regular:** μ(A) = sup{μ(K) : K ⊆ A, K compact}
- **Outer regular:** μ(A) = inf{μ(U) : U ⊇ A, U open}

The **Riesz representation theorem** (for measures): every positive linear functional Λ on Cₒ(X) (continuous functions vanishing at infinity) has the form Λ(f) = ∫f dμ for a unique regular Borel measure μ.

### Which Theorems Need σ-finiteness?

- **Fubini/Tonelli:** require σ-finiteness for uniqueness of the product measure
- **Radon-Nikodym:** requires σ-finiteness of both μ and ν
- **MCT, DCT, Fatou:** hold for all measure spaces
