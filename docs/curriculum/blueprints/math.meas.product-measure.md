## Component 1 — Learning Objectives

- LO1: Construct the PRODUCT $\sigma$-algebra $\mathcal{F}_1\otimes\mathcal{F}_2$ (generated by measurable rectangles $A\times B$) and the PRODUCT MEASURE $\mu_1\otimes\mu_2(A\times B)=\mu_1(A)\mu_2(B)$, recognizing this as the natural measure-theoretic generalization of `math.calc.double-integrals`'s own $dA=dx\,dy$ — area on rectangles multiplying, extended to abstract measure spaces.
- LO2: State FUBINI'S THEOREM precisely: if $\iint|f|\,d(\mu_1\otimes\mu_2)<\infty$ (the ABSOLUTE integral is FINITE), then the iterated integrals equal the double integral AND equal each other (order of integration doesn't matter) — recognizing this ABSOLUTE-INTEGRABILITY hypothesis as essential, not optional.
- LO3 (orientation level — full counterexample and Tonelli-proof machinery deferred): recognize, without full derivation, that TONELLI'S theorem relaxes Fubini's hypothesis for NON-NEGATIVE $f$ (order of integration may ALWAYS be switched, with no finiteness check needed — both orders simply agree, possibly both equal to $+\infty$), while for functions taking BOTH signs without absolute integrability, Fubini's conclusion can genuinely FAIL — different iteration orders can give different, finite answers.

## Component 2 — Prerequisite Check

Assumes mastery of `math.meas.lebesgue-integral` ($\int f\,d\mu$ as a supremum over simple functions, generalizing the Riemann integral).

## Component 3 — Core Explanation

**The product measure generalizes `math.calc.double-integrals`'s $dx\,dy$ to abstract measure spaces**: `math.calc.double-integrals` computes $\iint_Df(x,y)\,dA$ using $dA=dx\,dy$, implicitly relying on ordinary Lebesgue measure on $\mathbb{R}^2$ being the PRODUCT of $1$-dimensional Lebesgue measure on each axis: a rectangle's area is literally width times height, $\mu(A\times B)=\mu_1(A)\mu_2(B)$. The product measure construction ABSTRACTS this exact idea to ANY two measure spaces $(\Omega_1,\mathcal{F}_1,\mu_1)$ and $(\Omega_2,\mathcal{F}_2,\mu_2)$: define the product $\sigma$-algebra generated by measurable rectangles $A\times B$, and the product measure by $\mu_1\otimes\mu_2(A\times B)=\mu_1(A)\mu_2(B)$ — literally the same "multiply the pieces" idea, generalized beyond ordinary area.

**Fubini's theorem requires ABSOLUTE integrability — this hypothesis is essential, not a formality**: Fubini's theorem says: IF $\iint|f|\,d(\mu_1\otimes\mu_2)<\infty$ (the integral of the ABSOLUTE VALUE is finite), THEN the iterated integrals (integrate over one variable first, then the other) EQUAL the double integral, AND equal each other regardless of order. This absolute-integrability check is not a minor technical footnote — WITHOUT it, switching integration order can genuinely change the answer, because positive and negative contributions can be summed in different orders to reach different (even divergent-vs-convergent) results.

**Tonelli relaxes this for non-negative functions; without it, order can genuinely matter (orientation level)**: for NON-NEGATIVE $f$ (no sign issues to worry about), TONELLI'S theorem guarantees the order of integration may ALWAYS be freely switched — no absolute-integrability check needed, since both iterated integrals simply agree (possibly both equal to $+\infty$, which is still a valid, consistent answer). But for functions taking BOTH signs, WITHOUT Fubini's absolute-integrability hypothesis holding, the two iteration orders can give DIFFERENT finite answers — a genuine failure mode, not merely a theoretical curiosity, deferred to full development beyond this concept's core scope.

## Component 4 — Worked Examples

**Example 1 (LO1 — the product measure as ordinary area, generalized, breaking MC-1)**: for $A=[0,2]$ and $B=[0,3]$ (intervals on the real line, each with ordinary Lebesgue measure $\mu_1(A)=2$, $\mu_2(B)=3$), the product measure gives $\mu_1\otimes\mu_2(A\times B)=\mu_1(A)\mu_2(B)=2\times3=6$ — EXACTLY the area of the rectangle $[0,2]\times[0,3]$, matching `math.calc.double-integrals`'s own $dA=dx\,dy$ computation directly. This confirms the abstract product-measure construction, applied to ordinary $1$-dimensional Lebesgue measure on each factor, reproduces exactly the familiar area computation.

**Example 2 (LO2 — Fubini's absolute-integrability hypothesis and its role, breaking MC-2)**: for $f(x,y)=xye^{-(x^2+y^2)}$ on $[0,\infty)\times[0,\infty)$: checking $\iint|f|\,dA=\iint xye^{-(x^2+y^2)}\,dA$ (already non-negative here) $=\left(\int_0^\infty xe^{-x^2}\,dx\right)\left(\int_0^\infty ye^{-y^2}\,dy\right)=\frac12\times\frac12=\frac14<\infty$ — FINITE, so Fubini applies. This GUARANTEES the iterated integral in either order (integrate over $x$ first, then $y$; or $y$ first, then $x$) gives the SAME answer, $\frac14$ — a guarantee that specifically REQUIRED verifying the absolute-integrability condition first, not something automatically true for any function.

**Example 3 (LO3, orientation level — a case where switching order genuinely fails, breaking MC-3)**: the classic counterexample $f(x,y)=\dfrac{x^2-y^2}{(x^2+y^2)^2}$ on $[0,1]\times[0,1]$ (with $f(0,0)$ suitably defined) has $\int_0^1\left(\int_0^1f(x,y)\,dy\right)dx=\frac{\pi}{4}$ but $\int_0^1\left(\int_0^1f(x,y)\,dx\right)dy=-\frac{\pi}{4}$ — GENUINELY DIFFERENT answers depending on integration order, because $\iint|f|\,dA=\infty$ (absolute integrability FAILS here). This is not merely a theoretical warning: switching order without checking absolute integrability first can produce a demonstrably WRONG answer, confirming Fubini's hypothesis is doing essential work, not a formality to skip.

## Component 5 — Teaching Actions

### Teaching Action A01 — Product Measure Is "Multiply the Pieces," Generalized Beyond Area (Primitive P11: Representation Shift)

State: "you already compute area as width times height in `math.calc.double-integrals` — the product measure is exactly this same idea, abstracted to work for ANY two measure spaces, not just ordinary lengths." Walk Example 1's direct rectangle-area computation matching the abstract product-measure formula.

- **MC-1 hook**: ask "is the product measure construction an unrelated new idea, disconnected from the familiar 'area equals width times height' computation from `math.calc.double-integrals`?" — a "yes" answer reveals MC-1 (missing that the product measure is exactly this same multiplication idea, generalized to abstract measure spaces).

### Teaching Action A02 — Fubini's Absolute-Integrability Check Is Essential Work, Not a Formality (Primitive P28: Conflict Evidence)

Present Example 2's direct evidence: verifying $\iint|f|<\infty$ was an explicit, necessary step BEFORE concluding the iterated integrals agree. State: "checking absolute integrability isn't a box to tick mechanically — it's the specific hypothesis that GUARANTEES switching integration order is safe, and without checking it, you can't trust the conclusion."

- **MC-2 hook**: ask "can you always switch the order of integration in a double integral, as long as both iterated integrals happen to be individually computable?" — a "yes" answer reveals MC-2 (missing that Fubini's absolute-integrability hypothesis is required to guarantee the orders genuinely agree).

### Teaching Action A03 — Without Absolute Integrability, Order Can Genuinely Change the Answer (Primitive P06: Contrast Pair)

Contrast Example 2's Fubini-guaranteed order-independence (both orders give $\frac14$) against Example 3's genuine order-DEPENDENCE ($+\pi/4$ vs. $-\pi/4$) when absolute integrability fails. State: "this isn't a theoretical edge case that never comes up in practice — it's a concrete demonstration that skipping the absolute-integrability check can lead you to a genuinely wrong, order-dependent answer."

- **MC-3 hook**: ask "if a function fails the absolute-integrability check, does switching integration order still typically give the same answer anyway, just without a formal guarantee?" — a "yes" answer reveals MC-3 (missing that the two orders can give genuinely DIFFERENT, both individually finite, answers in this failure case).

### Teaching Action A04 — Mastery Gate (Primitive P91)

- **P77 (4-problem set)**:
  1. Compute the product measure of $[1,4]\times[2,5]$ using ordinary $1$-dimensional Lebesgue measure on each factor.
  2. For $f(x,y)=e^{-x-y}$ on $[0,\infty)\times[0,\infty)$, verify $\iint|f|<\infty$ and state what Fubini's theorem then guarantees.
  3. Explain, referencing the classic counterexample, what can go wrong when switching integration order for a function that fails absolute integrability.
  4. State Tonelli's theorem, and explain how its hypothesis (non-negativity) differs from Fubini's hypothesis (absolute integrability).

(a) Using `math.calc.double-integrals`'s own iterated-integral technique, set up and evaluate this integral in one order. (c) A colleague, working with a DIFFERENT, sign-changing function elsewhere in their analysis, switches integration order without checking absolute integrability first, assuming 'it always works for smooth-looking functions' — evaluate this assumption, citing the classic counterexample's lesson."
