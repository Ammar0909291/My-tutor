## Component 1 — Misconception Register

### MC-SIMILAR-SPELLING-MEANS-SAME-FAMILY
- **Trigger signal:** Student groups words as a "word family" based purely on surface spelling similarity rather than genuine shared root/base meaning (e.g., groups "car," "care," and "card" together because they all start with "car-," when they share no actual meaning connection — unlike "act," "action," "active," "reaction," which genuinely share the root meaning "to do").
- **Conflict evidence [P28]:** "You grouped 'car,' 'care,' and 'card' together because they all start with 'car.' Does 'care' (to look after something) have anything to do with 'car' (a vehicle) or 'card' (a piece of paper)? Now look at 'act,' 'action,' 'reaction' — do those share an actual meaning connection?"
- **Bridge text [P30]:** "A true word family shares a common BASE WORD or ROOT with a related MEANING, not just similar-looking letters at the start. 'Act,' 'action,' 'active,' 'reaction' all relate to the idea of 'doing' — that's a genuine word family. 'Car,' 'care,' and 'card' just happen to share letters by coincidence, with no meaning connection at all."
- **Replacement text [P31]:** "Check for a genuine shared MEANING connection through a common base/root, not just similar spelling at the start of words."
- **Discrimination pairs [P33]:**
  - "act, action, active, reaction" (true word family — shared meaning "to do") vs. "car, care, card" (spelling-similar only, no shared meaning).
  - "sign, signal, signature, design" (shared root meaning related to "marking/indicating") vs. "sin, sing, sing-along" (spelling overlap, no shared root meaning).
- **s6_path:** "Looking for shared letters is a very reasonable first strategy — but the real test of a word family is whether the words share an actual MEANING connection through a common root, not just how they look."

### MC-WORD-FAMILY-MEMBERS-ARE-ALWAYS-SAME-WORD-CLASS
- **Trigger signal:** Student assumes all members of a word family must be the same part of speech (e.g., assumes if "act" is a verb, every related word in its family must also be a verb), missing that word families typically span MULTIPLE word classes built from the same root (act = verb, action = noun, active = adjective, actively = adverb).
- **Conflict evidence [P28]:** "You said 'act' is a verb, so 'action' must also be a verb. But in the sentence 'His action surprised everyone,' is 'action' doing a verb's job or a noun's job?"
- **Bridge text [P30]:** "Word families typically include words from DIFFERENT word classes, all built from the same root — 'act' (verb), 'action' (noun), 'active' (adjective), 'actively' (adverb) are all members of the same family despite belonging to different grammatical categories. The shared meaning connection matters, not matching word class."
- **Replacement text [P31]:** "Expect word-family members to span different word classes (verb, noun, adjective, adverb forms) built from the same root — don't assume they must all match the first member's word class."
- **Discrimination pairs [P33]:**
  - "act" (verb) / "action" (noun) / "active" (adjective) / "actively" (adverb) — same family, four different word classes.
  - "create" (verb) / "creation" (noun) / "creative" (adjective) / "creatively" (adverb) — same pattern.
- **s6_path:** "It's a reasonable assumption that related words should behave the same grammatically — but word families are specifically defined by shared root MEANING across different word-class 'costumes,' which is actually a very useful pattern for building vocabulary efficiently once you notice it."

## Component 2 — Prerequisite Diagnostic Block

**PD-1 [P41] — Word-recognition readiness**
Prompt: "What does the word 'act' mean?"
- Pass: correctly recognizes and defines the base word, confirming genuine (not just decoded) recognition (per `eng.vocab.word-recognition`).
- Fail → [P52]: brief word-recognition refresher before proceeding. Persistent failure → route to eng.vocab.word-recognition for reteaching.

## Component 3 — Concrete Anchor [P06]

**Anchor scene — The word-family tree**

> Draw a simple tree with one root word at the base (e.g., "act"). Add branches for each related word-class form (action, active, actively, reaction), discussing how each branch still connects back to the same root meaning ("to do") despite looking and functioning differently. Then draw a SEPARATE, unconnected set of similar-LOOKING words (car, care, card) off to the side, explicitly showing they do NOT belong on this tree because they share no meaning connection. This anchors word families as meaning-rooted trees, explicitly contrasted with mere spelling lookalikes.

## Component 4 — Conceptual Development Sequence

**TA-1 — Identifying a Genuine Word Family from a Root [C]**

Present a root word (e.g., "happy") and its family members (happiness, happily, unhappy); student confirms the shared meaning connection across all forms.

**TA-2 — Distinguishing True Families from Spelling Lookalikes [C]**

Present mixed sets including both true word families and spelling-similar-but-unrelated words; student sorts correctly, directly targeting MC-SIMILAR-SPELLING-MEANS-SAME-FAMILY.

**TA-3 — Word Class Variation Within a Family [C→P]**

Present a word family (create/creation/creative/creatively); student identifies each member's word class, directly targeting MC-WORD-FAMILY-MEMBERS-ARE-ALWAYS-SAME-WORD-CLASS by requiring the student to recognize multiple different classes within one family.

**TA-4 — Generating Word Family Members [P]**

Given a root word, student generates as many family members as they can (e.g., "help" → helpful, helpless, helper, unhelpful), reinforcing productive use of the pattern.

**TA-5 — Using Word Family Knowledge to Infer Meaning [P]**

Present an unfamiliar word that is a member of an otherwise-known word family (e.g., student knows "create" but not "creativity"); student uses the known root's meaning to infer the unfamiliar word's meaning, demonstrating the practical vocabulary-expansion value of this concept.

## Component 5 — Worked Examples

**WE-1 (Foundational — genuine word family)**

> "Do 'happy,' 'happiness,' and 'unhappy' belong to the same word family? Why?"

Target response: yes — they all share the root meaning related to "happy," despite different forms/word classes.

**WE-2 (Foundational — spelling lookalike rejection)**

> "Do 'sign' and 'sing' belong to the same word family?"

Target response: no — despite similar spelling, they share no meaning connection ("sign" relates to marking/indicating; "sing" relates to music/voice).

**WE-3 (Intermediate — word class variation)**

> "In the word family 'create, creation, creative, creatively,' what word class is each one?"

Target response: "create" (verb), "creation" (noun), "creative" (adjective), "creatively" (adverb) — same family, four different classes.

## Component 6 — Mastery Probe Set

**MP-1 [P36] — Genuine word family, novel root**
"Do 'help,' 'helpful,' and 'helper' belong to the same word family?"
*Target:* yes — all share the root meaning related to "help."

**MP-2 [P36] — Spelling lookalike rejection, novel pair**
"Do 'fine' and 'finish' belong to the same word family?"
*Target:* no — despite similar spelling at the start, no shared meaning connection.

**MP-3 [P36] — Word class variation, novel family**
"In the word family 'decide, decision, decisive,' what word class is each one?"
*Target:* "decide" (verb), "decision" (noun), "decisive" (adjective).

**MP-4 [P36] — Generating family members, novel root**
"Give me as many word-family members of 'act' as you can."
*Target:* generates several, e.g., "action," "active," "actively," "reaction," "activate," "actor."

**MP-5 [P36] — Using word families to infer meaning**
"You know the word 'kind.' What might 'kindness' mean, based on the word family?"
*Target:* correctly infers "kindness" relates to the quality of being kind, using the shared root meaning.

```
  ↓

  ↓

[P89] Retrieval schedule: 1 day, 3 days, 7 days
[P91] Mastery gate: 75%; re-route to Component 3 anchor if below
```

**Protocol routing:** S0 (secure on word recognition, new to the word-family concept): dwell on TA-1's clear, unambiguous root/family examples before introducing the spelling-lookalike contrast. S1 (already recognizes some morphological patterns intuitively): use TA-2/TA-3 to build explicit, transferable vocabulary for an already-present pattern-recognition skill. S6 (frustrated by needing to check meaning rather than relying on the simpler spelling-matching strategy): validate that the meaning-check is a slightly more effortful but far more reliable strategy, worth the extra step. S9 (L1 uses a very different morphological system, e.g. an isolating language with little/no derivational morphology, or a language where affixes work very differently from English prefixes/suffixes): expect this concept's underlying assumption (that words are built from shared roots plus modifying affixes) to require more foundational explanation for this learner, since the very idea of "morphological family" may not map cleanly from their L1 structure.

## Component 8 — Adaptive Flags

- **Meaning connection is the defining test, not spelling**: every family-identification task should require an explicit meaning check, directly countering the natural shortcut of grouping by shared letters alone.
- **Word class diversity within a family is expected, not exceptional**: explicitly teach that family members typically span verb/noun/adjective/adverb forms — this is a core, useful feature of the pattern, not a complication to work around.
- **Generative practice reinforces productive vocabulary use**: TA-4's family-member-generation task builds active, productive vocabulary skill beyond passive recognition — prioritize this alongside receptive identification tasks.
- **Morphological-system transfer for L1s with different word-formation patterns**: for S9 learners whose L1 doesn't build words the way English does (root + prefix/suffix), provide more foundational scaffolding on the very concept of "a family of related word-forms" before assuming the pattern-recognition skill transfers readily.
