## Component 1 — Cognitive Map

### Prior Knowledge Anchors
- **math.found.set-theory**: sets and subsets; power set; unions and intersections (finite and arbitrary/indexed families); ∅ and the whole set; set-builder notation
- *(cross-link, not prerequisite)* **math.real.metric-space**: open balls; metric-open sets; the observation (that blueprint's P76) that d₂ and d∞ generate identical open sets on ℝ²

### Target Knowledge State
Student can state the three topology axioms (∅,X∈τ; arbitrary unions of members of τ are in τ; finite intersections of members of τ are in τ); verify whether a proposed collection of subsets is a topology on a finite set; work with the standard extreme examples (discrete topology = all subsets, indiscrete/trivial topology = {∅,X}); explain why intersections are restricted to FINITE while unions may be arbitrary (with the ℝ counterexample ∩ₙ(−1/n,1/n)={0}); recognise the metric topology (open sets generated by balls) as the motivating example; and understand that a topology is exactly the data needed to define convergence and continuity without any distance function.

### Conceptual Obstacles
1. Symmetrizing the axioms — assuming arbitrary intersections must also be allowed (or only finite unions); the asymmetry is essential, and the canonical counterexample ∩ₙ₌₁^∞(−1/n, 1/n)={0} (not open in ℝ) shows infinite intersections of open sets can fail to be open
2. Believing "open" has intrinsic meaning — a set is open ONLY relative to a chosen topology τ; the same subset of X can be open in one topology and not in another (in the discrete topology EVERY subset is open; in the indiscrete, almost none are)
3. Assuming every topology comes from a metric — the indiscrete topology on a two-point set has no metric generating it (a metric always separates distinct points into disjoint balls); topologies genuinely generalize metric spaces, not just repackage them

## Component 2 — Misconception Registry

| ID | Label | Description | Trigger |
|----|-------|-------------|---------|
| MC-1 | ARBITRARY-INTERSECTIONS-ALLOWED | Student states or applies the axioms with arbitrary intersections permitted (or restricts unions to finite); accepts collections as topologies that fail the actual axioms, or rejects valid ones | Axiom recall; verification tasks on infinite families; the (−1/n,1/n) example |
| MC-2 | OPEN-IS-ABSOLUTE | Student treats openness as an intrinsic property of a set ("(0,1) is open, period") rather than relative to a declared topology; confused when the same set is open in τ₁ but not in τ₂ | Finite-set topologies; comparing discrete vs indiscrete; subset questions like "is {a} open?" |
| MC-3 | EVERY-TOPOLOGY-IS-METRIC | Student assumes every topology arises from some metric (topology = metric spaces restated); cannot accept the indiscrete topology or finite non-discrete topologies as genuine, metric-free structures | Non-metrizable examples; "which metric gives this topology?" questions |

**Foundational Misconception:** MC-2 (OPEN-IS-ABSOLUTE) — the entire subject rests on openness being DECLARED by τ rather than inherited from geometry; a student who thinks "open" is absolute cannot verify topologies on finite sets, cannot compare topologies, and misreads every subsequent definition (continuity, convergence) that quantifies over "all open sets."

## Component 4 — Protocol A (Main Teaching Sequence)

### Teaching Action A01 — Topologies on a Three-Point Set

**Primitive:** P11 REPRESENTATION SHIFT
**Purpose:** Make the axioms operational through exhaustive finite verification; establish that openness is declared, not discovered; justify the finite-intersection restriction with the classical ℝ counterexample

---

**[P11 — REPRESENTATION SHIFT]**

**Stage C — Concrete (two candidates on X={a,b,c}):**

A **topology** on X is a collection τ of subsets — the sets we DECLARE to be "open." The declaration must obey three closure rules (stated fully in Stage A). Test two candidates:

**Candidate 1:** τ₁ = {∅, {a}, {a,b}, X}

- ∅∈τ₁ ✓ and X∈τ₁ ✓
- Unions: {a}∪{a,b}={a,b} ✓; {a}∪X=X ✓; {a,b}∪X=X ✓; any union involving ∅ changes nothing ✓
- Intersections: {a}∩{a,b}={a} ✓; {a}∩X={a} ✓; {a,b}∩X={a,b} ✓; with ∅ gives ∅ ✓

All checks pass → **τ₁ IS a topology.** (Note: {b}, {c}, {b,c} are simply NOT open here — that is allowed; τ need not contain every subset.)

**Candidate 2:** τ₂ = {∅, {a}, {b}, X}

- ∅, X ∈ τ₂ ✓
- Union: {a}∪{b}={a,b}. Is {a,b}∈τ₂? **No.** ✗

One failed union → **τ₂ is NOT a topology.** (Repair: adding {a,b} yields τ₂′={∅,{a},{b},{a,b},X}, which does pass all checks.)

**Stage P — Pictorial (three standard topologies on {a,b,c} as lattice diagrams):**

```
 Indiscrete τ={∅,X}      Intermediate τ₁            Discrete τ=all 8 subsets
        X                     X                          X
        |                     |                        / | \
        |                   {a,b}                 {a,b}{a,c}{b,c}
        |                     |                        \ | /  (all pairs)
        ∅                    {a}                    {a} {b} {c}
                              |                        \ | /
                              ∅                          ∅
   2 open sets            4 open sets                8 open sets
   (coarsest)                                        (finest)
```

Every set X carries at least these two extremes: the **indiscrete** (trivial) topology {∅,X} and the **discrete** topology (the whole power set). Between them, many intermediate topologies can exist — τ₁ is one of them.

**Stage A — Abstract (the axioms, and the finite-intersection restriction):**

**Definition:** A **topological space** is a pair (X, τ) where τ is a collection of subsets of X (called the **open sets**) satisfying:

| # | Axiom |
|---|-------|
| T1 | ∅ ∈ τ and X ∈ τ |
| T2 | Any union of members of τ is in τ (arbitrary — finite, countable, or uncountable index sets) |
| T3 | Any FINITE intersection of members of τ is in τ (equivalently: the intersection of any TWO) |

**Why is T3 restricted to finite while T2 is not?** The axioms are modeled on the open sets of ℝ, where the asymmetry is forced:

$$\bigcap_{n=1}^{\infty}\left(-\tfrac{1}{n},\, \tfrac{1}{n}\right) = \{0\}$$

Each interval (−1/n, 1/n) is open in ℝ, but the intersection is a single point — and {0} is not open in ℝ (no interval around 0 fits inside it). Infinite intersections can squeeze open sets down to non-open ones. Unions have no such failure: a union of intervals can only get more room, never less. The axioms encode exactly what survives.

---

**[P49 — ADAPTIVE CHECKPOINT]**

**Prompt:** Is τ = {∅, {a}, {b,c}, X} a topology on X={a,b,c}? Check all three axioms.

**CORRECT:** T1: ∅,X∈τ ✓. T2: {a}∪{b,c}=X∈τ ✓ (other unions give X or repeat members) ✓. T3: {a}∩{b,c}=∅∈τ ✓ (other intersections give ∅ or repeat members) ✓. **Yes — a topology** (X partitioned into two open pieces).
→ "Correct. This topology splits X into two 'clopen' blocks — a disconnected space, in later language." → Proceed to A02.

**PARTIAL:** Student checks T1 and T2 but skips intersections, or checks only one pair.
→ "T3 still needs checking. The only nontrivial pair is {a}∩{b,c}=∅ — and ∅∈τ ✓. With small finite collections, list every pair systematically; a single missing union or intersection disqualifies the whole collection (as with τ₂ in the lesson)."

**INCORRECT:** Student rejects it because "{b} and {c} aren't open" (MC-2).
→ "A topology does not need to make every subset — or every singleton — open. τ only has to be CLOSED under the operations on the sets it does contain. Here every union and intersection of members of τ lands back in τ, so τ is a topology. {b} not being open is a feature of THIS topology, not a violation."

**NO_RESPONSE:** → "Walk the axioms: T1 — ∅ and X are both listed ✓. T2 — the only new union to test is {a}∪{b,c}=X, which is in τ ✓. T3 — the only new intersection is {a}∩{b,c}=∅, in τ ✓. All three axioms hold: τ is a topology."

### Teaching Action A02 — Openness Is Relative; Topologies Beyond Metrics

**Primitive:** P06 CONTRAST PAIR
**Purpose:** Break MC-2 with the same set open/not-open under different topologies; break MC-3 with a provably non-metrizable topology; position the metric topology as one source of topologies among several

---

**[P06 — CONTRAST PAIR]**

**Contrast 1 — The same set, three verdicts (MC-2):**

Is {a} open in X={a,b,c}? The question is MEANINGLESS until a topology is named:

| Topology on {a,b,c} | Is {a} open? |
|---------------------|--------------|
| Discrete (all subsets) | ✓ yes |
| τ₁={∅,{a},{a,b},X} | ✓ yes |
| Indiscrete {∅,X} | ✗ no |

"Open" is a membership claim about τ — never an intrinsic property of the set. Even in ℝ: (0,1) is open in the standard topology, but on the SET ℝ equipped with the indiscrete topology, (0,1) is not open. The habit to build: every openness claim silently carries "…in the topology τ."

**Contrast 2 — Where topologies come from: metric vs declared:**

*From a metric:* given (X,d), declare U open iff every x∈U has a ball B(x,r)⊆U. This collection always satisfies T1–T3 (T2: balls witness membership pointwise; T3: for x in both U and V, take the smaller of the two radii). This is the **metric topology** — the standard topology on ℝⁿ arises this way.

*Declared directly:* the topologies of A01 on {a,b,c} were written down with no metric in sight. Both routes produce legitimate topological spaces; the axioms don't remember where τ came from.

**Contrast 3 — A topology NO metric can produce (MC-3):**

Claim: the indiscrete topology {∅, X} on X={a,b} is **not metrizable**.

*Proof:* Suppose a metric d gave this topology. Since a≠b, r:=d(a,b)>0 (metric axiom M1). The ball B(a, r/2) is open in the metric topology, contains a, and excludes b (since d(a,b)=r>r/2) — so it is a nonempty open set that is neither ∅ nor X. But the indiscrete topology has NO such set. Contradiction. ∎

Metrics always separate distinct points with disjoint balls; the indiscrete topology refuses all separation. Hence topological spaces form a strictly LARGER world than metric spaces — the generalization is genuine (this is the content of the KG description: "generalizes 'neighborhood' beyond metric spaces").

---

**[P49 — ADAPTIVE CHECKPOINT]**

**Prompt:** On X={1,2,3,4}, let τ={∅, {1,2}, {3,4}, X}. (a) Verify τ is a topology. (b) Is {1} open in (X,τ)? (c) Could some metric on X generate exactly this τ?

**CORRECT:** (a) T1 ✓; unions: {1,2}∪{3,4}=X∈τ ✓; intersections: {1,2}∩{3,4}=∅∈τ ✓ — topology. (b) {1}∉τ, so {1} is NOT open in this topology (though it would be in the discrete one). (c) No: in any metric space, each singleton {x} of a FINITE space is open (take r smaller than the least distance to the other points — the ball is {x}), so a metric on a finite set always generates the discrete topology. τ is not discrete → not metrizable.
→ "Correct on all three — (c) is exactly the finite-space version of the non-metrizability argument." → Proceed to A03.

**PARTIAL:** Student gets (a) and (b) but answers (c) "yes, some clever metric might work" (MC-3).
→ "Try to build one: whatever d you choose, let r be the minimum distance between distinct points (finitely many pairs, so r>0 exists). Then B(1, r) = {1} is open in the metric topology. But {1}∉τ. No metric can avoid making singletons open on a finite set — so no metric generates τ."

**INCORRECT:** Student answers (b) "yes, {1} is open because single points are open" (MC-2).
→ "Openness is membership in the DECLARED τ. Scan the list: τ={∅,{1,2},{3,4},X} — {1} is not there, so {1} is not open in THIS space. Your intuition comes from the discrete (or metric) picture, which is a different topology on the same set."

**NO_RESPONSE:** → "(a) Only two nontrivial checks: {1,2}∪{3,4}=X ∈τ and {1,2}∩{3,4}=∅ ∈τ — topology ✓. (b) Is {1} in the list τ? No → not open. (c) Any metric on a finite set makes every singleton a ball of small radius, forcing the discrete topology — but τ isn't discrete, so no metric produces it."

### Teaching Action A03 — Convergence and Continuity Without Distance

**Primitive:** P03 ANALOGY BRIDGE
**Purpose:** Bridge from the metric/ε-δ formulations to their pure open-set restatements, showing that τ alone carries the analytic concepts; demonstrate consequences in the extreme topologies

---

**[P03 — ANALOGY BRIDGE]**

**Source (metric language):** In a metric space, xₙ→x means: every ball B(x,ε) eventually contains the whole tail of the sequence. Continuity of f at a: every ε-ball around f(a) has a δ-ball around a mapped into it.

**Bridge:** Notice both definitions consume balls only through one property — being open sets containing the point. Replace "ball around x" by "open set containing x" (a **neighborhood**):

**Target (topological language):**

- **Convergence:** xₙ → x in (X,τ) ⟺ for every open U∋x, there is N with xₙ∈U for all n≥N.
- **Continuity:** f:(X,τ_X)→(Y,τ_Y) is continuous ⟺ for every open V∈τ_Y, the preimage f⁻¹(V) is open in X.

No distance appears. Any topology — declared or metric-generated — now supports limits and continuous maps. In metric spaces the new definitions agree with the old ε-δ ones (balls are open, and every open set contains a ball around each of its points), so nothing is lost; generality is gained.

**Where the analogy extends and where it strains (the extremes as stress tests):**

- *Discrete topology:* every set is open, so {x} is a neighborhood of x — convergence forces xₙ=x eventually (matching the discrete metric); EVERY function OUT of a discrete space is continuous (every preimage is open for free).
- *Indiscrete topology:* the only neighborhood of any point is X itself, which contains every sequence tail trivially — so **every sequence converges to every point**. Limits lose uniqueness. This is not a paradox but a signal: topologies this coarse carry too little information to separate points (the future axiom "Hausdorff" restores uniqueness).
- ⚠ Strain: metric intuition ("limits are unique," "sequences detect everything") is a property of NICE topologies, not of the definition itself.

---

**[P49 — ADAPTIVE CHECKPOINT]**

**Prompt:** Let X={a,b,c} carry τ₁={∅,{a},{a,b},X} and let Y={0,1} carry the discrete topology. Define f:X→Y by f(a)=0, f(b)=0, f(c)=1. Is f continuous? Use the preimage criterion.

**CORRECT:** Open sets of Y: ∅,{0},{1},Y. Preimages: f⁻¹(∅)=∅ ✓; f⁻¹({0})={a,b}∈τ₁ ✓; f⁻¹({1})={c} — is {c}∈τ₁? No ✗. One preimage fails → **f is not continuous.**
→ "Correct. A single non-open preimage decides it. (Swap f(c) to 0 and f becomes constant-0 — continuous, since preimages are only ∅, X.)" → Proceed to A04.

**PARTIAL:** Student checks f⁻¹({0}) only and declares continuity.
→ "The criterion quantifies over EVERY open set of Y. You still owe {1}: f⁻¹({1})={c}, and {c}∉τ₁. Not continuous. Checklist habit: enumerate all of τ_Y (here just four sets) and test each preimage."

**INCORRECT:** Student computes images instead of preimages (checks whether f({a}) is open in Y).
→ "Continuity looks BACKWARD: preimages of open sets must be open — images play no role (continuous maps can have non-open images). Compute f⁻¹(V)={x∈X: f(x)∈V} for each open V⊆Y: f⁻¹({0})={a,b} ✓ open, f⁻¹({1})={c} ✗ not in τ₁. Not continuous."

**NO_RESPONSE:** → "List τ_Y (discrete on two points): ∅,{0},{1},{0,1}. Pull each back: f⁻¹(∅)=∅ ∈τ₁; f⁻¹({0})={a,b} ∈τ₁; f⁻¹({1})={c} ∉τ₁ — stop: f is not continuous; f⁻¹(Y)=X ∈τ₁ (moot)."

### Teaching Action A04 — Mastery Gate (P91)

**Primitive:** P91 = P77→P55→P76→P55→P75→P55→P74→P55→P78
**Purpose:** Assess axiom fluency, relative-openness understanding, and the metric→topology cross-link

**[P77 — MULTI-PROBLEM SET]** *(4 problems)*

**Problem 1:** State the three axioms of a topology, and explain with a specific example in ℝ why intersections are restricted to finite families.

*Solution:* T1: ∅,X∈τ; T2: arbitrary unions stay in τ; T3: finite intersections stay in τ. Restriction justified by ∩ₙ₌₁^∞(−1/n,1/n)={0}: each factor is open in ℝ, but {0} contains no open interval around 0, hence is not open — infinite intersections can destroy openness.

**Problem 2:** Determine which of the following are topologies on X={a,b}: (i) {∅,{a},X}, (ii) {∅,{a},{b}}, (iii) {∅,{a},{b},X}.

*Solution:* (i) Topology ✓ (the "Sierpiński space": all unions/intersections land in the list). (ii) NOT a topology: X={a,b}∉τ — fails T1 (also {a}∪{b}={a,b} missing). (iii) Topology ✓ — this is the discrete topology on X (all four subsets).

**Problem 3:** On X={1,2,3}, is τ={∅,{1},{2},{1,2},X} a topology? Verify every union and intersection.

*Solution:* T1 ✓. Unions: {1}∪{2}={1,2}∈τ ✓; {1}∪{1,2}={1,2} ✓; {2}∪{1,2}={1,2} ✓; anything∪X=X ✓. Intersections: {1}∩{2}=∅ ✓; {1}∩{1,2}={1} ✓; {2}∩{1,2}={2} ✓; anything∩X=itself ✓. **Topology ✓.**

**Problem 4:** In (X,τ) with X={1,2,3}, τ={∅,{1},{2},{1,2},X}: is the set {3} open? Is {1,3} open? Name one topology on X in which both ARE open.

*Solution:* {3}∉τ → not open; {1,3}∉τ → not open. In the **discrete topology** (all 8 subsets) both are open. (Openness is relative to the declared τ.)

**[P76 — TRANSFER PROBE]** *(P76_mode: cross-link probe — math.real.metric-space)*

**Prompt:** Let (X,d) be a metric space and define
$$\tau_d = \{\,U \subseteq X : \text{for every } x\in U \text{ there is } r>0 \text{ with } B(x,r)\subseteq U\,\}.$$

(a) Prove that τ_d satisfies all three topology axioms (T1, T2, T3). For T3, explain where FINITENESS of the intersection is used, and why the argument breaks for infinite intersections.

(b) On ℝ² it is known that d∞(x,y) ≤ d₂(x,y) ≤ √2·d∞(x,y). Using (a) and this sandwich, explain why τ_{d₂} = τ_{d∞} — and state in one sentence what this says about the relationship between metrics and topologies (many metrics, one topology).

**Expected solution:**

(a)
- **T1:** ∅∈τ_d vacuously (no points to check); X∈τ_d since every ball is a subset of X.
- **T2:** Let U=∪ᵢUᵢ with each Uᵢ∈τ_d, and x∈U. Then x∈Uᵢ₀ for some i₀, which supplies B(x,r)⊆Uᵢ₀⊆U. Arbitrary index sets cause no trouble — one witness suffices. ✓
- **T3:** Let x∈U₁∩⋯∩Uₙ. Each Uₖ supplies rₖ>0 with B(x,rₖ)⊆Uₖ. Take r=min(r₁,…,rₙ): then B(x,r)⊆every Uₖ, so B(x,r)⊆∩Uₖ. ✓ **Finiteness is used exactly at the minimum**: finitely many positive radii have a positive minimum. For infinitely many sets the infimum of the rₖ can be 0 (e.g. Uₙ=(−1/n,1/n) at x=0 forces rₙ≤1/n, inf=0), and no single positive radius survives — which is precisely why ∩ₙ(−1/n,1/n)={0} fails to be open.

(b) From d∞≤d₂: every d₂-ball of radius r sits inside the d∞-ball of radius r, i.e. B₂(x,r)⊆B∞(x,r); from d₂≤√2·d∞: B∞(x, r/√2)⊆B₂(x,r). So any set that is open with respect to one metric's balls admits, around each of its points, a ball of the other metric too — membership in τ_{d₂} and τ_{d∞} impose the same requirement, hence τ_{d₂}=τ_{d∞}. **Conclusion:** distinct metrics can generate the identical topology — the topology retains only the "neighborhood structure" and forgets the numerical distances, which is exactly why topology is the coarser, more general theory.

## Component 5 — Protocol B (Misconception Repair)

### Repair Action B01 — ARBITRARY-INTERSECTIONS-ALLOWED (MC-1)

**[P27 — MISCONCEPTION NAMING]**
"Stating the axioms with arbitrary intersections (or only finite unions) is ARBITRARY-INTERSECTIONS-ALLOWED. The real axioms are asymmetric: unions arbitrary, intersections finite — and the asymmetry is forced by ℝ itself."

**[P41 — MISCONCEPTION DETECTOR]**
Diagnostic: "Recite the topology axioms. Do infinite intersections of open sets have to be open?"
- MC-1 response: "Yes — unions and intersections both, any size."

**[P64 — CONCEPTUAL SHIFT]**
"Run the killer example: in ℝ, each (−1/n, 1/n) is open. Intersect over ALL n: a point in every interval must satisfy |x|<1/n for every n, so x=0 — the intersection is {0}. Open? An open set containing 0 must contain an interval (−r, r), but {0} contains no interval. Not open. So the axiom CANNOT demand arbitrary intersections and still describe ℝ. Mechanism to remember: intersections shrink the room; finitely many shrinkages leave a positive minimum radius, infinitely many can squeeze it to zero. Unions only add room — no size limit needed."

Practice: Show ∩ₙ(0, 1+1/n)=(0,1] in ℝ and explain which endpoint breaks openness.

### Repair Action B02 — OPEN-IS-ABSOLUTE (MC-2)

**[P27 — MISCONCEPTION NAMING]**
"Treating openness as an intrinsic property of a set is OPEN-IS-ABSOLUTE. 'Open' means 'a member of the declared τ' — the same set can be open in one topology and not in another."

**[P41 — MISCONCEPTION DETECTOR]**
Diagnostic: "On X={a,b,c} with τ={∅,{a,b},X}: is {a} open?"
- MC-2 response: "Yes, single points are open," or "yes, {a} is a small simple set."

**[P64 — CONCEPTUAL SHIFT]**
"Openness is a LOOKUP, not a judgment: scan τ={∅,{a,b},X} — is {a} on the list? No → not open, end of story. The same {a} IS open in the discrete topology on the same X. The full habit: every claim 'U is open' expands to 'U∈τ' for a named τ. This is also why one set X supports many different spaces: (X, discrete), (X, indiscrete), (X, τ₁) are three different topological spaces sharing the same points."

Practice: For X={a,b,c}, list all topologies you can build containing exactly 3 open sets ({∅, S, X} for each proper nonempty S — verify each is a topology), and note that each makes a different single set open.

### Repair Action B03 — EVERY-TOPOLOGY-IS-METRIC (MC-3)

**[P27 — MISCONCEPTION NAMING]**
"Assuming every topology comes from some metric is EVERY-TOPOLOGY-IS-METRIC. Metric spaces generate topologies, but not every topology is so generated — the generalization is strict."

**[P41 — MISCONCEPTION DETECTOR]**
Diagnostic: "The indiscrete topology on {a,b} — which metric produces it?"
- MC-3 response: attempts to construct one (e.g. d(a,b)=0, or 'a very small distance').

**[P64 — CONCEPTUAL SHIFT]**
"No metric works, provably. d(a,b)=0 is illegal (M1 forces distinct points to positive distance). So r=d(a,b)>0, and B(a, r/2)={a} is open in the metric topology — a nonempty open set that isn't X. The indiscrete τ={∅,X} has no such set. Contradiction: not metrizable. Moral: metrics always SEPARATE points; topologies are free not to. That freedom is the extra generality — 'metrizable' becomes a special property a space may or may not have, studied later, rather than a given."

Practice: Adapt the argument to show the topology {∅,{a},X} on {a,b} (Sierpiński) is also non-metrizable (a metric would make {b} open too).

## Component 8 — Teaching Notes

**Structural decisions:**
- h=5 → standard structure (3 main TAs + gate)
- bloom=understand → V-4 = N/A (no P07 required; axiom verification and structural-understanding tasks)
- CPA_entry = C for expert learner: exhaustive finite verification on {a,b,c} in A01 before ℝ or metrics appear

**Key teaching insight:** Finite examples first is the deliberate inversion of the usual "open intervals → abstract axioms" route. On {a,b,c}, openness is TRANSPARENTLY a declaration (the set is on the list or it isn't), which inoculates against MC-2 before ℝ-intuition can install it. The metric topology then arrives in A02/P76 as one generator of topologies rather than as the concept's secret identity.

**The asymmetry (T2 vs T3):** This is the single most examined subtlety of the definition. Both the counterexample (∩(−1/n,1/n)={0}) and the MECHANISM (finite minima of radii stay positive; infinite infima can vanish) should be produced by the student, not just recognized — P77 Problem 1 and P76(a) both demand it.

**Sequencing note (metric-space cross-link):** math.real.metric-space is a cross-link, not a prerequisite; this blueprint's A01–A02 are self-contained on finite sets. A student arriving WITHOUT the metric blueprint can still complete everything except the P76 probe comfortably; a student arriving WITH it will recognize P76 as the formal completion of that blueprint's closing observation. Teachers should sequence metric-space first when possible.

**P76 design:** The probe is the payoff of the pair of blueprints: (a) proves the ball-generated collection is a topology, with the finiteness usage made explicit (T3's minimum argument) — connecting the axiom asymmetry to a concrete proof step; (b) reuses the d₂/d∞ sandwich to conclude topology-equality, forcing the "many metrics, one topology" insight that DEFINES the subject's level of abstraction.
