## Component 1 — Learning Objectives

- LO1: Define $N$ as **normal** in $G$ (written $N\trianglelefteq G$) iff $gNg^{-1}=N$ for EVERY $g\in G$, and verify normality directly for a specific subgroup by computing $gNg^{-1}$ for representative elements $g$.
- LO2: Recognize that $gNg^{-1}=N$ is equivalent to $gN=Ng$ as SETS (left and right cosets coincide) — a genuinely more permissive condition than requiring $g$ to commute with every individual element of $N$ — and recognize that while EVERY subgroup of an abelian group is automatically normal, a NON-abelian group can genuinely have subgroups that fail to be normal.
- LO3: State that kernels of group homomorphisms are ALWAYS normal, and that normality is EXACTLY the condition required for the coset multiplication $(aN)(bN)=abN$ to be **well-defined** — independent of which representatives $a,b$ are chosen for their respective cosets — directly motivating why normal subgroups, specifically, are singled out for the quotient-group construction.

## Component 2 — Prerequisite Check

Assumes mastery of `math.abst.coset` (the left coset $aH=\{ah:h\in H\}$; the equality criterion $aH=bH$ iff $a^{-1}b\in H$; cosets partition $G$ into equal-sized, disjoint blocks; the index $[G:H]=|G|/|H|$).

## Component 3 — Core Explanation

**Normal: $gNg^{-1}=N$ for every $g$, not just some**: a subgroup $N\le G$ is **normal** ($N\trianglelefteq G$) if $gNg^{-1}=N$ for EVERY $g\in G$ — conjugating $N$ by any group element returns $N$ itself (as a set), never a genuinely different subgroup. This is a real, checkable condition — not every subgroup satisfies it.

**Set-level equality, not element-wise commutation**: $gNg^{-1}=N$ is equivalent to $gN=Ng$ (left cosets equal right cosets, as SETS). This does NOT require $g$ to commute with each individual element of $N$ ($gn=ng$ for every specific $n$) — it only requires that for each $n\in N$, SOME element $n'\in N$ satisfies $gn=n'g$ (the set $\{gn : n\in N\}$ equals the set $\{ng : n\in N\}$, even if the individual pairings differ from simple commutation). In an ABELIAN group, every subgroup is automatically normal (since $gn=ng$ trivially for every $n$, by the group's own commutativity) — but in a NON-abelian group, this guarantee disappears entirely, and genuine failures of normality can and do occur.

**Why normality matters: kernels, and well-defined quotient multiplication**: the kernel of any group homomorphism is ALWAYS a normal subgroup — a key structural fact connecting normality to homomorphisms. More fundamentally, normality is EXACTLY the condition needed to make the coset multiplication $(aN)(bN):=abN$ well-defined: if $N$ is normal, this product does not depend on which representatives $a,a'$ (with $aN=a'N$) or $b,b'$ are chosen. If $N$ is NOT normal, different valid representatives of the same cosets can produce genuinely DIFFERENT products — making "multiply the cosets" an ill-defined operation, and the quotient group construction fails outright.

## Component 4 — Worked Examples

Throughout, work in $D_3$ (the symmetries of an equilateral triangle, $|D_3|=6$): rotations $e,r,r^2$ ($r^3=e$) and reflections $s,sr,sr^2$ ($s^2=e$), with the defining relation $sr=r^{-1}s=r^2s$ (a reflection followed by a rotation equals the inverse rotation followed by the reflection) — the standard smallest non-abelian group.

**Example 1 (LO1 — a genuinely normal subgroup)**: Let $H=\{e,r,r^2\}$ (the rotation subgroup). Check $sHs^{-1}=sHs$ (since $s^{-1}=s$): $ses=s^2=e$; $srs = (sr)s = (r^2s)s = r^2s^2=r^2$; $sr^2s = s(r\cdot r)s = (sr)(rs)$ — using $sr=r^2s$ and (from the same relation, rearranged) $rs=sr^2$: $(sr)(rs)=(r^2s)(sr^2)=r^2(s^2)r^2=r^2\cdot e\cdot r^2=r^4=r$ (since $r^3=e$). So $sHs=\{e,r^2,r\}=H$ exactly — confirming $H$ IS normal.

**Example 2 (LO1/LO2 — a genuinely NON-normal subgroup, breaking MC-1)**: Let $K=\{e,s\}$ (generated by one reflection). Check $rKr^{-1}$: $rer^{-1}=e$; $rsr^{-1}$ — using $rs=sr^2$ (from the relation above) and $r^{-1}=r^2$: $rsr^{-1}=(rs)r^2=(sr^2)r^2=sr^4=sr$ (since $r^4=r$). So $rKr^{-1}=\{e,sr\}$ — but $K=\{e,s\}$, and $sr\ne s$ (since $r\ne e$) — so $rKr^{-1}\ne K$. **$K$ is NOT normal**, despite being a perfectly legitimate subgroup of $D_3$ — direct, computed proof that not every subgroup of a non-abelian group is normal.

**Example 3 (LO2/LO3 — set-level equality without element-wise commutation, and quotient multiplication breaking down without normality)**: For $H$ (normal) from Example 1: $sH=\{s,sr,sr^2\}$ and $Hs=\{es,rs,r^2s\}=\{s,rs,r^2s\}$. Using $sr=r^2s$ and $sr^2=(sr)r=(r^2s)r=r^2(sr)=r^2(r^2s)=r^4s=rs$: $sH=\{s,r^2s,rs\}$, exactly matching $Hs=\{s,rs,r^2s\}$ — the SETS are equal, even though $s$ does NOT commute with $r$ element-wise ($sr=r^2s\ne rs$, since $r^2\ne r$). Now for $K$ (NOT normal): the coset $rK=\{r,rs\}$ has two valid representatives, $r$ and $rs$. Computing "$(rK)(rK)$" using representative $r$: $r\cdot r=r^2$, giving $r^2K=\{r^2,r^2s\}$. Using representative $rs$ instead: $(rs)(rs) = r(sr)s = r(r^2s)s = r^3s^2=e\cdot e=e$, giving $eK=K=\{e,s\}$. Since $r^2K\ne K$ (they are different sets), the "product" of the SAME coset $rK$ with itself gives two genuinely DIFFERENT answers depending on which representative was used — the multiplication is **ill-defined** precisely because $K$ is not normal.

## Component 5 — Teaching Actions

### Teaching Action A01 — Verifying Normality Directly (Primitive P11: Representation Shift)

State: "normal means conjugating the ENTIRE subgroup by any element gives back the same subgroup — not a different one." Work Example 1's full computation confirming $sHs=H$.

### Teaching Action A02 — Not Every Subgroup Is Normal (Primitive P28: Conflict Evidence)

Present Example 2's direct computation: $rKr^{-1}=\{e,sr\}\ne\{e,s\}=K$. State: "$K$ is a completely legitimate subgroup of $D_3$ — it just isn't a NORMAL one. In a non-abelian group, this genuinely happens, and you have to check, not assume."

- **MC-1 hook**: ask "is every subgroup of every group automatically normal?" — a "yes" answer reveals MC-1 (over-generalizing from the abelian case, where this happens to be true, to all groups).

### Teaching Action A03 — Set Equality Isn't Element-Wise Commutation; Non-Normality Breaks the Quotient (Primitive P06: Contrast Pair)

Present Example 3's two parts: first, $sH=Hs$ as SETS despite $s$ and $r$ not commuting individually ($sr\ne rs$); then, $K$'s two different representatives of the coset $rK$ giving genuinely different "products," $r^2K$ versus $K$ itself. State: "$gN=Ng$ only ever claims the SETS match — not that $g$ personally commutes with each element. And when a subgroup fails to be normal, this isn't just a technicality — it means coset multiplication itself stops making sense, which is exactly why the quotient group construction needs normality."

- **MC-2 hook**: ask "does gN=Ng mean g commutes with every individual element of N?" — a "yes" answer reveals MC-2 (confusing set-level coset equality with full element-wise commutation).
- **MC-3 hook**: ask "does the coset multiplication (aN)(bN)=abN always give a consistent, well-defined answer, no matter which subgroup N is?" — a "yes" answer reveals MC-3 (missing that non-normal subgroups make this multiplication genuinely ill-defined, as Example 3's two conflicting products demonstrate).

### Teaching Action A04 — Mastery Gate (Primitive P91)

P91 = P77 → P55 → P76 → P55 → P75 → P55 → P74 → P55 → P78.

- **P77 (4-problem set)**:
  1. In $D_3$, verify $r^2Hr^{-2}=H$ for $H=\{e,r,r^2\}$ (using a rotation, rather than a reflection, as the conjugating element).
  2. In $D_3$, check whether $sr\cdot K\cdot(sr)^{-1}=K$ for $K=\{e,s\}$, using $(sr)^{-1}=r^{-1}s^{-1}=r^2s$, and state whether this confirms or further disproves $K$'s normality.
  3. Explain why every subgroup of an abelian group is automatically normal, referencing the definition $gNg^{-1}=N$.
  4. Explain, using Example 3's two different computed products for $rK\cdot rK$, why $K$'s failure to be normal makes the coset multiplication genuinely ill-defined, not just inconvenient.
- **P76 (Transfer Probe, mode = independence)**: "A company organizes its departments (subgroups of the full staff, under some abstract 'combination' operation representing joint projects) and wants to define a simplified 'department-level' organizational chart by treating each department as a single unit (a coset). (a) Explain, in your own words, why the company would want the 'combine two departments' operation to give the SAME resulting department-level answer regardless of which specific employee is chosen to represent each department. (b) Using this lesson's $K=\{e,s\}$ counterexample as an analogy, explain what could go wrong for the organizational chart if the departments-as-subgroups do not satisfy the normality condition. (c) Explain why, in a fully cooperative (abelian-like) company where every combination of employees is 'order-independent,' this problem would never arise in the first place."
- **P55 (Score)** after each stage.
- **P75 (Mastery Assessment against MAMR = 5/5)**.
- **P74 (Routing Decision)**: MAMR met → P78 (Completion). MAMR not met → route to the specific missed primitive's repair action before re-attempting P77.

## Component 6 — Misconception Registry

| ID | Label | Description | Severity |
|---|---|---|---|
| MC-1 | EVERY-SUBGROUP-ASSUMED-NORMAL | Believing every subgroup of every group is automatically normal, over-generalizing from the abelian case (where this happens to hold) to non-abelian groups in general | Foundational |
| MC-2 | GN-EQUALS-NG-CONFUSED-WITH-ELEMENT-WISE-COMMUTATION | Believing gN=Ng means g commutes with every individual element of N, rather than the weaker set-level equality of the two coset sets | Foundational |
| MC-3 | QUOTIENT-MULTIPLICATION-ASSUMED-ALWAYS-WELL-DEFINED | Believing coset multiplication (aN)(bN)=abN is always well-defined regardless of whether N is normal, missing that non-normal subgroups produce genuinely inconsistent products depending on the chosen representatives | Foundational |

### Protocol B — Repair Actions

- **B01 (targets MC-1)**: P27 (name it: "Every Subgroup Assumed Normal") → P41 (detect: ask a student whether every subgroup of a given non-abelian group must be normal, and check for a "yes") → P64 (conceptual shift: re-walk Example 2's direct computation showing $rKr^{-1}\ne K$, re-anchoring on "normality genuinely has to be checked in a non-abelian group — it is not automatic").
- **B02 (targets MC-2)**: P27 (name it: "gN Equals Ng Confused with Element-Wise Commutation") → P41 (detect: ask a student whether gN=Ng requires g to commute with each element of N individually, and check for a "yes") → P64 (conceptual shift: re-walk Example 3's $sH=Hs$ verification, showing the sets match despite $sr\ne rs$ individually, re-anchoring on "it's the SETS that must match — not every individual pairing").
- **B03 (targets MC-3)**: P27 (name it: "Quotient Multiplication Assumed Always Well-Defined") → P41 (detect: ask a student whether coset multiplication always gives a consistent answer regardless of the subgroup used, and check for a "yes") → P64 (conceptual shift: re-walk Example 3's two conflicting computations for $rK\cdot rK$ ($r^2K$ versus $K$), re-anchoring on "normality is exactly what prevents this kind of inconsistency — without it, the 'multiplication' isn't a genuine operation at all").

## Component 8 — Teaching Notes

- estimated_hours = 4 with an advanced/understand tag places this at a "3 TAs + gate" tier — A01 (verifying a normal subgroup), A02 (a genuinely non-normal counterexample), and A03 (the set-vs-element-wise distinction, plus the quotient-breakdown demonstration) each target a distinct LO, appropriate for a concept whose core difficulty is conceptual precision (exactly what the definition does and does not require) rather than heavy computation.
- All three worked examples deliberately use the SAME concrete group, $D_3$ (the smallest non-abelian group), with the SAME two subgroups ($H$, normal; $K$, not normal) reused throughout — letting a single, fully worked concrete setting carry every LO and every misconception-breaking contrast, rather than introducing fresh groups for each objective.
- $D_3$ was chosen deliberately over an abstract/symbolic-only treatment specifically because MC-1 (every subgroup is normal) can ONLY be genuinely refuted with a non-abelian example — an abelian group would make every subgroup trivially normal, providing no counterexample at all.
