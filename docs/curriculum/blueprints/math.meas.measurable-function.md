## Component 1 — Learning Objectives

- LO1: Define $f:(X,\mathcal{F})\to(Y,\mathcal{G})$ (between two measurable spaces) as **measurable** iff $f^{-1}(E)\in\mathcal{F}$ for **every** $E\in\mathcal{G}$ — the preimage of every measurable target set is itself measurable in the source — and verify this condition directly for a simple function.
- LO2: State the practical **Borel-measurability test**: for $f:\mathbb{R}\to\mathbb{R}$ (or into $\mathbb{R}$ with its Borel σ-algebra), it suffices to check that $f^{-1}((a,b))$ (the preimage of every OPEN INTERVAL, or more generally every open set) is measurable — NOT needing to check every conceivable Borel set individually.
- LO3: State that **every continuous function on $\mathbb{R}$ is automatically Borel measurable**, and explain WHY this follows directly from the topological definition of continuity (preimages of open sets are open, hence measurable) — correctly recognizing measurability as a genuinely WEAKER requirement than continuity (a measurable function need not be continuous).

## Component 2 — Prerequisite Check

Assumes mastery of `math.meas.sigma-algebra` (the σ-algebra $\mathcal{F}$ on a set $X$, and the measurable space $(X,\mathcal{F})$).

## Component 3 — Core Explanation

A function $f:(X,\mathcal{F})\to(Y,\mathcal{G})$ between two **measurable spaces** is **measurable** iff:
$$f^{-1}(E) \in \mathcal{F} \quad \text{for every } E\in\mathcal{G}$$
— every measurable subset of the TARGET, pulled back through $f$, lands as a measurable subset of the SOURCE. (This directly parallels the topological definition of a continuous function — "preimages of open sets are open" — with "measurable" replacing "open" throughout.)

**The practical test (Borel case)**: checking $f^{-1}(E)\in\mathcal{F}$ for EVERY possible Borel set $E\subseteq\mathbb{R}$ (there are uncountably many) would be impractical directly — but it SUFFICES to check just the OPEN INTERVALS (or open sets generally), since the Borel σ-algebra is GENERATED by open sets, and preimage-taking respects the σ-algebra operations (a general, provable fact: if $f^{-1}(E)$ is measurable for every $E$ in a GENERATING collection, it's automatically measurable for every set in the whole σ-algebra generated by that collection).

**Continuous implies measurable (Borel)**: every continuous function $f:\mathbb{R}\to\mathbb{R}$ is automatically Borel measurable — this follows DIRECTLY from the topological definition of continuity itself: $f$ continuous means $f^{-1}(U)$ is OPEN for every open $U$, and every open set is (trivially) measurable (it's IN the Borel σ-algebra, which is generated by open sets) — so the practical test above is satisfied automatically.

**Measurability is genuinely weaker than continuity**: NOT every measurable function is continuous — e.g., a step function (like the indicator function of an interval) is measurable (its preimages of open sets are always measurable, even though the function itself has jumps and isn't continuous) but is NOT continuous. Measurability is a much larger, more permissive class of functions than continuity.

## Component 4 — Worked Examples

**Example 1 (LO1 — verifying measurability directly)**: Let $X=\{1,2,3,4\}$ with $\mathcal{F}=\{\emptyset,\{1,2\},\{3,4\},X\}$, and $f:X\to\mathbb{R}$ defined by $f(1)=f(2)=0$, $f(3)=f(4)=1$. Check measurability against $\mathcal{G}=$ Borel sets on $\mathbb{R}$: for $E=(-1,0.5)$ (an interval containing 0 but not 1), $f^{-1}(E)=\{1,2\}\in\mathcal{F}$ ✓. For $E=(0.5,1.5)$, $f^{-1}(E)=\{3,4\}\in\mathcal{F}$ ✓. Since $f$'s image only takes the two values 0 and 1, and both preimages land in $\mathcal{F}$, $f$ is measurable relative to this $\mathcal{F}$.

**Example 2 (LO2 — the open-interval sufficiency test)**: For $f(x)=x^2$ (a continuous function on $\mathbb{R}$), rather than checking EVERY Borel set, check just open intervals: $f^{-1}((a,b))$ for $0<a<b$ is $(-\sqrt b,-\sqrt a)\cup(\sqrt a,\sqrt b)$ — a union of two open intervals, itself measurable (open sets are Borel measurable, and finite/countable unions of measurable sets are measurable, by the σ-algebra closure properties). Since this pattern holds for every open interval (verifiable case by case for different sign/position combinations), the sufficiency test confirms $f(x)=x^2$ is Borel measurable — without ever needing to check a single non-interval Borel set directly.

**Example 3 (LO3 — measurable but not continuous, breaking MC-1)**: Let $g(x) = \begin{cases}0 & x<0\\1 & x\geq0\end{cases}$ (a step function, the indicator of $[0,\infty)$). Is $g$ continuous? NO — it jumps discontinuously at $x=0$. Is $g$ measurable? YES — check preimages of open intervals: $g^{-1}((-0.5,0.5)) = (-\infty,0)$ (an open set, hence measurable); $g^{-1}((0.5,1.5))=[0,\infty)$ (a closed set — but closed sets ARE Borel measurable too, being complements of open sets); every other interval's preimage is similarly either $\emptyset$, $(-\infty,0)$, $[0,\infty)$, or all of $\mathbb{R}$ — ALL measurable. So $g$ is measurable DESPITE being discontinuous — directly demonstrating that measurability is a strictly weaker (more permissive) condition than continuity.

## Component 5 — Teaching Actions

### Teaching Action A01 — The Measurability Definition, and the Practical Open-Set Test (Primitive P11: Representation Shift)

Recall the continuity definition (preimages of open sets are open). State: "measurability is EXACTLY this idea, generalized — replace 'open' with 'measurable' throughout, and allow the source and target to be any measurable spaces, not just $\mathbb{R}$ with its usual topology." Work Example 1's small finite-space verification directly.

Shift representation to the practical Borel test: work Example 2's open-interval-by-open-interval check for $f(x)=x^2$, emphasizing that checking just the GENERATING open sets suffices — a genuine computational shortcut, not an approximation.

- **MC-1 hook**: ask "since every continuous function is measurable, does that mean every measurable function must also be continuous?" — a "yes" answer reveals MC-1 (believing measurability and continuity are equivalent, rather than recognizing measurability as the strictly WEAKER, more inclusive condition).

### Teaching Action A02 — Measurable-But-Not-Continuous, and Why Open Sets Suffice (Primitive P06: Contrast Pair)

**Contrast 1 (targets MC-1)**: Work Example 3's step function explicitly — confirmed measurable (every relevant preimage lands as a measurable set) but confirmed NOT continuous (a genuine jump at $x=0$). State the rule precisely: "continuous $\Rightarrow$ measurable is a TRUE, one-directional implication — the converse is FALSE; there are many measurable functions, like step functions, that are not continuous at all."

**Contrast 2**: Revisit WHY checking only open intervals suffices for the full Borel test (Example 2) — state the general principle: "if $\mathcal{G}$ is GENERATED by a smaller collection of sets (like open intervals generating the Borel σ-algebra), and $f^{-1}$ behaves well with respect to countable unions/complements (a fact inherited directly from $\mathcal{F}$ being a σ-algebra), then checking the generating collection is enough — this dramatically reduces the practical verification burden from 'infinitely many Borel sets' to 'just the open intervals.'"

### Teaching Action A03 — Mastery Gate (Primitive P91)

P91 = P77 → P55 → P76 → P55 → P75 → P55 → P74 → P55 → P78.

- **P77 (4-problem set)**:
  1. Verify that $f(x)=3x+1$ is Borel measurable by checking the preimage of a general open interval $(a,b)$.
  2. Consider the function $h(x)=|x|$ (continuous, but with a "corner" at 0, not differentiable there). Explain why $h$ is nonetheless Borel measurable, citing the relevant general fact.
  3. Define a step function taking value 2 on $(-\infty,3)$ and value 5 on $[3,\infty)$. Verify it is measurable despite its discontinuity at $x=3$, by checking preimages of a few representative open intervals.
  4. Explain precisely why "continuous implies measurable" is a one-directional implication, using the step-function example to demonstrate the converse fails.
- **P76 (Transfer Probe, mode = cross-link probe against `math.prob.random-variable`)**: "Recall from your `math.prob.random-variable` lesson that a random variable is defined as a function $X:\Omega\to\mathbb{R}$ assigning a real number to each outcome — and that lesson's own notes explicitly flagged that the FULL formal definition actually requires $X$ to be a MEASURABLE function (a requirement that lesson deferred as 'not yet available' at the time it was authored). (a) Using THIS lesson's measurability definition, state precisely what additional condition $X:\Omega\to\mathbb{R}$ must satisfy, beyond simply being 'a function,' to formally qualify as a random variable (hint: $X^{-1}(E)$ must be an event — i.e., must lie in the σ-algebra of events on $\Omega$ — for every Borel set $E\subseteq\mathbb{R}$). (b) Explain why, for a FINITE or countable sample space $\Omega$ (like a die roll, from that lesson's own worked example) where every subset counts as an event, this measurability condition is automatically satisfied for ANY function $X:\Omega\to\mathbb{R}$ — meaning the measurability requirement is invisible/automatic at that lesson's level, only becoming a genuine additional constraint for uncountable sample spaces."

## Component 6 — Misconception Registry

| ID | Label | Description | Severity |
|---|---|---|---|
| MC-1 | MEASURABLE-ASSUMED-EQUIVALENT-TO-CONTINUOUS | Believing measurability and continuity are equivalent conditions, rather than recognizing measurability as strictly weaker (every continuous function is measurable, but not vice versa) | Foundational |
| MC-2 | ALL-BOREL-SETS-ASSUMED-TO-REQUIRE-INDIVIDUAL-CHECKING | Believing verifying measurability requires checking the preimage of every individual Borel set, rather than recognizing checking just a generating collection (like open intervals) suffices | Moderate |
| MC-3 | MEASURABILITY-DEFINITION-DIRECTION-REVERSED | Stating the measurability condition backwards — checking $f(A)\in\mathcal{G}$ for $A\in\mathcal{F}$ (forward image) rather than the correct $f^{-1}(E)\in\mathcal{F}$ for $E\in\mathcal{G}$ (preimage) | Moderate |

### Protocol B — Repair Actions

- **B01 (targets MC-1)**: P27 (name it: "Measurable-Continuous Equivalence Assumption") → P41 (detect: ask whether every measurable function must be continuous, checking against the step-function example) → P64 (conceptual shift: work through Example 3's explicit verification, showing a genuinely discontinuous function passing the measurability test cleanly).
- **B02 (targets MC-2)**: P27 (name it: "Individual Borel-Set Checking Overestimation") → P41 (detect: ask a student how many sets must be checked to confirm Borel measurability, watching for "infinitely many, individually") → P64 (conceptual shift: re-anchor on the generating-set principle — checking open intervals alone suffices, since the σ-algebra structure propagates automatically).
- **B03 (targets MC-3)**: P27 (name it: "Measurability Direction Reversed") → P41 (detect: ask a student to state the measurability definition and check whether they write $f(A)\in\mathcal{G}$ instead of $f^{-1}(E)\in\mathcal{F}$) → P64 (conceptual shift: re-derive by direct analogy with the continuity definition — "preimages of [open/measurable] sets," never forward images, which need not behave well at all).
