## Component 1 — Learning Objectives

- LO1: Define a **principal ideal** ($(a)=\{ra:r\in R\}$, generated by a single element) and a **Principal Ideal Domain (PID)** as an integral domain where EVERY ideal is principal — and PROVE every Euclidean Domain is a PID, via the minimal-norm-element argument, as a direct consequence of `math.abst.euclidean-domain`'s own already-mastered division-with-remainder property.
- LO2: Apply the PID-specific structural facts — **every prime ideal is maximal**, and **irreducibles equal primes** — verifying both concretely in $\mathbb{Z}$ (a familiar PID), and recognizing these are genuinely PID-SPECIFIC guarantees, not automatic facts in every integral domain.
- LO3: Recognize the converse of LO1 FAILS — not every PID is a Euclidean Domain — citing the celebrated example $\mathbb{Z}\left[\frac{1+\sqrt{-19}}2\right]$ (a PID with no possible Euclidean norm) as a landmark separating case.

## Component 2 — Prerequisite Check

Assumes mastery of `math.abst.euclidean-domain` (an integral domain with a Euclidean norm $N$ allowing division with remainder; the Euclidean Algorithm; $\mathbb{Z}$ and $F[x]$ as concrete instances).

## Component 3 — Core Explanation

**Every Euclidean Domain is a PID — proven via the minimal-norm-element argument**: given ANY ideal $I$ in a Euclidean Domain $R$ (with $I\ne\{0\}$), take $d\in I$ with the SMALLEST norm $N(d)$ among nonzero elements of $I$. For any $a\in I$, Euclidean division gives $a=qd+r$ with $r=0$ or $N(r)<N(d)$. Since $I$ is an ideal, $r=a-qd\in I$ too — but $d$ was chosen with MINIMAL norm among $I$'s nonzero elements, so $N(r)<N(d)$ is impossible unless $r=0$. Hence $a=qd\in(d)$ — every element of $I$ is a multiple of $d$, so $I=(d)$ is principal. This holds for EVERY ideal, so $R$ is a PID — directly derived from the Euclidean property already mastered, not an independently-verified new fact.

**PID structure: every prime ideal is maximal; irreducibles equal primes**: these are genuinely PID-SPECIFIC guarantees — in a PID, if $(p)$ is a nonzero prime ideal, it is automatically MAXIMAL (no ideal properly contains it except the whole ring), and every IRREDUCIBLE element is automatically PRIME (and vice versa). Neither guarantee holds in a general integral domain — some domains have irreducible elements that fail to be prime (a genuinely different, weaker situation), making the PID structure theorem a real, non-vacuous strengthening.

**The converse fails — PID does not imply Euclidean Domain**: "every ED is a PID" does not reverse. The celebrated example $R=\mathbb{Z}\left[\frac{1+\sqrt{-19}}2\right]$ is a PID (every ideal genuinely is principal) yet admits NO Euclidean norm function whatsoever — a deep, historically significant result. This confirms PID is a strictly BROADER class of rings than Euclidean Domain, not merely an equivalent rephrasing.

## Component 4 — Worked Examples

**Example 1 (LO1 — the minimal-norm proof concretely in ℤ, breaking MC-1)**: Let $I$ be a nonzero ideal of $\mathbb{Z}$, using the norm $N(a)=|a|$ (already mastered from `math.abst.euclidean-domain`). Let $d\in I$ have the smallest $|d|$ among nonzero elements of $I$ (say $d=6$, if $I$ happens to be generated by multiples of $6$). For any $a\in I$ (say $a=18$): Euclidean division gives $18=3(6)+0$, remainder $0$ — so $18=3d\in(d)$. For $a=20$: $20=3(6)+2$, remainder $2$; since $2\in I$ (as $I$ is an ideal and $20,18\in I$ forces $20-18=2\in I$) and $2<6=d$'s norm, this WOULD contradict $d$'s minimality — so if $I$ genuinely contains $20$, $d$ could not have been $6$ after all (a smaller element like $2$ would exist in $I$). This concrete back-and-forth confirms: every element of a genuine minimal-norm ideal must reduce to a multiple of $d$ with zero remainder, exactly the general argument's mechanism.

**Example 2 (LO2 — prime ideal maximality and irreducible-equals-prime in ℤ, breaking MC-2)**: In $\mathbb{Z}$ (a PID), is $7$ irreducible (no factorization into two non-units)? Yes. Is $(7)$ a MAXIMAL ideal? Every ideal of $\mathbb{Z}$ is $(n)$ for some $n$, and $(7)\subseteq(n)$ exactly when $n\mid7$ — so $n\in\{\pm1,\pm7\}$, giving only the TRIVIAL containments $(7)\subseteq(1)=\mathbb{Z}$ and $(7)\subseteq(7)$ itself — no ideal sits strictly between $(7)$ and $\mathbb{Z}$, confirming $(7)$ IS maximal, exactly matching the PID structure theorem. Contrast (stated, not derived in full): in a GENERAL integral domain (not necessarily a PID), an irreducible element need NOT be prime — a genuinely weaker situation the PID guarantee specifically rules out.

**Example 3 (LO3 — the PID/ED separation, breaking MC-3)**: $\mathbb{Z}\left[\frac{1+\sqrt{-19}}2\right]$ is a celebrated example (established via deep algebraic number theory, not re-derived here) of a ring that IS a PID — every one of its ideals is genuinely principal — yet admits NO possible Euclidean norm function at all, no matter how one might try to define "size" on it. This confirms PID is a strictly LARGER class than Euclidean Domain — "every ED is a PID" is a one-directional implication, with a genuine, historically significant gap on the other side.

## Component 5 — Teaching Actions

### Teaching Action A01 — Every Euclidean Domain Is a PID, Proven via Minimal Norm (Primitive P11: Representation Shift)

State: "PID status for a Euclidean Domain isn't a separate fact to verify — it's a direct, provable consequence of the minimal-norm-element argument, using exactly the division-with-remainder property already mastered." Work Example 1's concrete minimal-norm demonstration in $\mathbb{Z}$.

- **MC-1 hook**: ask "must PID status be verified independently for a Euclidean Domain, separate from its Euclidean property?" — a "yes" answer reveals MC-1 (missing that PID status is directly derived from the Euclidean property via the minimal-norm argument).

### Teaching Action A02 — PID Structure Is Not Automatic in Every Domain (Primitive P28: Conflict Evidence)

Present Example 2's direct evidence: $(7)$ is genuinely maximal in the PID $\mathbb{Z}$, confirmed by checking all possible containing ideals. State: "'every prime ideal is maximal' and 'irreducibles equal primes' are genuine PID-specific guarantees — general integral domains can fail to have irreducible elements that are prime."

- **MC-2 hook**: ask "do 'every prime ideal is maximal' and 'irreducibles equal primes' hold automatically in any integral domain, not just PIDs?" — a "yes" answer reveals MC-2 (missing that these are PID-specific structural facts, not general integral-domain facts).

### Teaching Action A03 — PID Is a Strictly Broader Class Than Euclidean Domain (Primitive P06: Contrast Pair)

Contrast the provable "every ED is a PID" (Example 1) against the celebrated counterexample $\mathbb{Z}\left[\frac{1+\sqrt{-19}}2\right]$ (a PID with NO Euclidean norm, Example 3). State: "the implication only goes one way — PID is a genuinely larger, strictly broader class of rings than Euclidean Domain."

- **MC-3 hook**: ask "are PID and Euclidean Domain equivalent classes of rings — every PID also automatically a Euclidean Domain?" — a "yes" answer reveals MC-3 (missing the genuine, historically significant separating counterexample).

### Teaching Action A04 — Mastery Gate (Primitive P91)

P91 = P77 → P55 → P76 → P55 → P75 → P55 → P74 → P55 → P78.

- **P77 (4-problem set)**:
  1. Explain, using the minimal-norm argument, why $\mathbb{R}[x]$ (a Euclidean Domain) must be a PID.
  2. In $\mathbb{Z}$, verify that $(11)$ is a maximal ideal, checking all possible containing ideals.
  3. State the relationship between "every ED is a PID" and "every PID is an ED" — which direction is proven, and which fails.
  4. Explain why the existence of $\mathbb{Z}\left[\frac{1+\sqrt{-19}}2\right]$ as a non-Euclidean PID confirms PID is a strictly larger class than ED, rather than just a differently-named equivalent class.
- **P76 (Transfer Probe, mode = independence)**: "An algebraist is studying a new ring $R$ and has verified it is a Euclidean Domain using a specific norm function. (a) Using this lesson's minimal-norm argument, explain why the algebraist can immediately conclude $R$ is also a PID, without any additional verification. (b) A colleague, studying a DIFFERENT ring $S$, has verified every ideal of $S$ is principal, and asks whether $S$ must therefore also be a Euclidean Domain. Using this lesson's PID/ED separation, explain why this conclusion does not follow automatically. (c) The colleague also assumes that since $S$ is a PID, every irreducible element of $S$ must automatically be prime, without further checking. Explain why this specific conclusion (unlike the ED-to-PID one) IS actually justified here, and what makes it different from assuming ED status transfers."
- **P55 (Score)** after each stage.
