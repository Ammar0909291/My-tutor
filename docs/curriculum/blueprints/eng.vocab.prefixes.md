## Component 1 — Misconception Register

### MC-A-PREFIX-ALWAYS-HAS-ONE-FIXED-MEANING
- **Trigger signal:** Student assumes a given prefix always contributes exactly the same meaning in every word, missing that some prefixes are polysemous (have multiple distinct meanings) or that the same letter sequence can sometimes NOT be a true prefix at all (e.g., assumes "re-" always means "again" — correct for "redo," "rewrite" — but incorrectly applies this to "relax" or "result," where "re-" is not a meaningful prefix at all, just a coincidental initial letter sequence).
- **Conflict evidence [P28]:** "You said 're-' always means 'again.' Does 'relax' mean to 'lax again'? Is there even a word 'lax' that 'relax' is built from?"
- **Bridge text [P30]:** "Most prefixes DO reliably contribute a consistent meaning (re- = again, un- = not, pre- = before) — but this only applies when the rest of the word is a genuine, recognizable base word. In words like 'relax' or 'result,' the letters 're-' at the start are NOT functioning as the meaningful prefix 're-' at all, because there's no base word like 'lax' or 'sult' being modified. Always check whether removing the suspected prefix leaves a real, meaningful base word."
- **Replacement text [P31]:** "Before assuming a prefix contributes its usual meaning, check whether removing it leaves a genuine, recognizable base word — if not, it's probably not functioning as that prefix at all."
- **Discrimination pairs [P33]:**
  - "redo" (re- + "do," a real base word, meaning "do again" — true prefix use) vs. "relax" (no real base word "lax" being modified — "re-" here is not a meaningful prefix).
  - "unhappy" (un- + "happy," a real base word, meaning "not happy") vs. "uncle" (no base word "cle" — "un-" here is not a meaningful prefix).
- **s6_path:** "It's a very reasonable pattern to expect once you notice how reliably 're-' or 'un-' work in many words — but always checking for a genuine base word underneath protects you from over-applying the pattern to words where it doesn't actually function that way."

### MC-PREFIXES-ONLY-NEGATE-MEANING
- **Trigger signal:** Student assumes all (or most) prefixes work like negation prefixes (un-, non-, dis-, in-/im-), reversing or negating the base word's meaning, missing that many prefixes add other kinds of meaning entirely — location/direction (sub-, super-, inter-, trans-), time/order (pre-, post-, re-), number (bi-, tri-, multi-), or degree (over-, under-).
- **Conflict evidence [P28]:** "You said prefixes make a word mean the opposite. Does 'submarine' mean 'not a marine'? What does 'sub-' actually add here?"
- **Bridge text [P30]:** "Negation prefixes (un-, non-, dis-, in-/im-) are one common category, but prefixes also express location (sub- = under, super- = above, inter- = between), time (pre- = before, post- = after), number (bi- = two, tri- = three), and degree (over- = too much, under- = too little) — negation is just one of several distinct meaning categories."
- **Replacement text [P31]:** "Check what CATEGORY of meaning a prefix adds — negation is common, but also consider location, time, number, and degree before assuming negation."
- **Discrimination pairs [P33]:**
  - "unhappy" (negation: un- = not) vs. "submarine" (location: sub- = under).
  - "disagree" (negation: dis- = not/opposite) vs. "prehistoric" (time: pre- = before).
- **s6_path:** "Negation prefixes are often taught first and are very memorable, which makes them feel like the main pattern — but prefixes covering location, time, number, and degree are just as common and important to recognize."

## Component 3 — Concrete Anchor [P06]

**Anchor scene — The prefix building blocks**

> Present a base-word block (e.g., "happy") and several prefix blocks (un-, re-, pre-) that can attach to it (only "un-" genuinely combines meaningfully here — "unhappy"). Physically attach a prefix block to the base and discuss the resulting meaning change. Then show a "trick" word (e.g., "uncle") where the "un-" letters are present but there's no meaningful base word "cle" to attach to — explicitly demonstrating the check-for-a-real-base-word test.

---

## Component 4 — Conceptual Development Sequence

**TA-1 — Negation Prefixes [C]**

Present base words with negation prefixes (unhappy, disagree, impossible, nonfiction); student identifies the prefix, the base word, and the resulting meaning change.

**TA-2 — Non-Negation Prefix Categories (Location, Time, Number, Degree) [C]**

Present words with prefixes from other categories (submarine, prehistoric, bicycle, overcook); student identifies the prefix's meaning category, directly targeting MC-PREFIXES-ONLY-NEGATE-MEANING.

**TA-3 — Checking for a Genuine Base Word [C→P]**

Present a mixed set of words with prefix-like letter sequences, some true (redo, unhappy) and some false (relax, uncle, prefer — where "pre-" doesn't clearly modify a recognizable base "fer"); student checks for a real base word before confirming prefix status, directly targeting MC-A-PREFIX-ALWAYS-HAS-ONE-FIXED-MEANING.

**TA-4 — Using Prefix Knowledge to Infer Meaning [P]**

Given an unfamiliar word containing a known prefix and a recognizable base word (e.g., "unbreakable," "international"), student infers the meaning by combining the prefix's category meaning with the base word's meaning.

**TA-5 — Generating Words with a Given Prefix [P]**

Given a prefix (e.g., "re-"), student generates several genuine words using it correctly (redo, rewrite, replay, reopen), reinforcing productive use of the pattern.

## Component 5 — Worked Examples

**WE-1 (Foundational — negation prefix)**

> "What does 'un-' mean in 'unhappy'? What's the resulting meaning?"

Target response: "un-" means "not"; "unhappy" means "not happy."

**WE-2 (Foundational — non-negation prefix)**

> "What does 'sub-' mean in 'submarine'? Is this negation?"

Target response: "sub-" means "under" (location) — not negation; "submarine" means "under the sea."

**WE-3 (Intermediate — checking for a genuine base word)**

> "Is 're-' functioning as a meaningful prefix in 'relax'? What about in 'redo'?"

Target response: in "relax," no — there's no base word "lax" being modified meaningfully in this context (recognizing "re-" here is coincidental, not the "again" prefix); in "redo," yes — "do" is a real base word, and "redo" means "do again."

## Component 6 — Mastery Probe Set

**MP-1 [P36] — Negation prefix, novel word**
"What does 'dis-' mean in 'disagree'?"
*Target:* "not" or "opposite of" — negation.

**MP-2 [P36] — Non-negation prefix, novel word**
"What does 'tri-' mean in 'triangle'? What category is this?"
*Target:* "three" — number category, not negation.

**MP-3 [P36] — Genuine base word check, novel example**
"Is 'pre-' a meaningful prefix in 'prevent'? Is there a recognizable base word 'vent' with a related meaning here?"
*Target:* correctly reasons that "prevent" doesn't clearly decompose into "pre-" + a transparent base word "vent" in modern English (this is a Latin-derived word where the historical roots aren't transparent to most learners) — recognizing the limits of prefix analysis for such words.

**MP-4 [P36] — Inferring meaning from a known prefix + base**
"What might 'unbreakable' mean, using the prefix 'un-' and the base word 'breakable'?"
*Target:* "not able to be broken" — correctly combines negation with the base word's meaning.

**MP-5 [P36] — Explain prefixes and their categories**
"In your own words: do all prefixes make a word negative, and how do you check if a prefix is really working as a prefix?"
*Target:* states or demonstrates that prefixes cover multiple meaning categories (negation, location, time, number, degree — not just negation), and that a genuine prefix requires a recognizable base word underneath it.

**Protocol routing:** S0 (secure on word families, new to prefixes specifically): dwell on TA-1's negation prefixes with clear, unambiguous examples before introducing other categories. S1 (already recognizes some prefixes intuitively): use TA-2/TA-3 to build explicit category-awareness and the base-word-check skill for an already-present pattern-recognition ability. S6 (frustrated by "trick" words that look like they have a prefix but don't): validate that this check is a genuine, valuable skill precisely because English has many such words, and that noticing the difference protects against overconfident misanalysis. S9 (L1 has a very different morphological system with no prefix-based word formation, e.g. an isolating or root-and-pattern language): expect this concept's underlying assumption (words built by adding meaningful pieces to a base) to require more foundational scaffolding, since the very idea of prefixation may not have a direct L1 parallel.

---

## Component 8 — Adaptive Flags

- **Base-word check is the essential discipline, not an afterthought**: always require verification that a genuine base word exists before confirming prefix status — this directly counters over-application of prefix patterns to coincidental letter sequences.
- **Multiple meaning categories, not just negation**: explicitly teach location, time, number, and degree prefixes alongside negation from early instruction, since negation prefixes' memorability shouldn't crowd out the other equally common categories.
- **Productive generation reinforces active vocabulary use**: TA-5's word-generation task builds active, productive skill beyond passive recognition — prioritize this alongside receptive identification tasks, paralleling the word-families concept's approach.
- **Morphological-system transfer for L1s without prefix-based word formation**: for S9 learners whose L1 doesn't build words via prefixation, provide more foundational scaffolding on the very concept of "meaningful word-building pieces" before assuming the pattern-recognition skill transfers readily.
